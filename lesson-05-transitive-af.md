# 第5课：及物动词句 AF——完整的主事焦点句型

## 本课目标

学完本课后，你将能够：
1. 用完整 AF 句型表达"X做Y"（两个参与者的句子）
2. 正确使用斜格标记 **to** 标记受事者
3. 理解 ko（主格）和 to（斜格）在 AF 句中的分工
4. 掌握 AF 句的两种词序变体
5. 认出格位三分系统（ko/no/to）的初步轮廓

## 核心知识

### 1. 从单论元到双论元

回顾第4课的单论元（不及物）句：

```
Mafoti'   kako.         "我睡觉。"
动词·AF   主语(主格)
```

当动词涉及**两个参与者**（动作发出者 + 动作承受者）时，阿美语引入斜格标记 **to**：

```
Komaen    ci Mayaw   to fonga.     "Mayaw吃地瓜。"
动词·AF   主事者(主格)  受事者(斜格)
```

[FACT] 在 AF 句中，**主事者**（动作执行者）用**主格**（ko/ci），**受事者**（动作承受者）用**斜格**（to）。p.33, 35

### 2. AF 及物句公式

```
┌─────────────────────────────────────────────────┐
│  V.AF  +  ko/ci  +  主事者  +  to  +  受事者     │
│  动词    主格标记   动作发出   斜格   动作承受     │
│  (mi-/    者(主语)  标记   者(非主语)             │
│  ma-/                                            │
│  <om>)                                           │
└─────────────────────────────────────────────────┘
```

| 阿美语 | 分析 | 汉语 |
|--------|------|------|
| **K**\<om\>**aen** **ci** Mayaw **to** fonga. | 吃·AF + ci+Mayaw(主格) + to+地瓜(斜格) | Mayaw吃地瓜。 |
| **Mi**'atal **ko** singsi **to** hemay. | 盛·AF + ko+老师(主格) + to+饭(斜格) | 老师在盛饭。 |
| **Mi**foting **ci** Panay **to** foting. | 捕鱼·AF + ci+Panay(主格) + to+鱼(斜格) | Panay在捕鱼。 |

[FACT] pp.33, 35-36

### 3. to 是什么？——斜格标记

[FACT] **to** 是普通名词的**斜格（Oblique）**标记。在 AF 句中最常见的功能是标记**受事者**（动作的承受者）。p.44

| 标记 | 格位 | AF句中的角色 | 名词类型 |
|------|------|------------|---------|
| **ko** | 主格 (NOM) | 主事者（动作发出者） | 普通名词 |
| **ci** | 主格 (NOM) | 主事者（动作发出者） | 专有人名单数 |
| **to** | 斜格 (OBL) | 受事者（动作承受者） | 普通名词 |

[COMPARISON] 汉语用**语序**区分主语和宾语（"Mayaw吃地瓜" vs "地瓜吃Mayaw"——语序决定谁吃谁）。阿美语用语序+**格位标记**双重手段：主格 ko/ci 标记主语，斜格 to 标记受事宾语。即使调换语序，标记仍然告诉你谁是谁。

这就是阿美语比汉语"多一层保险"的地方——标记系统让你即使不依赖语序也能判断语法关系。

### 4. 词序变体

[FACT] AF 双论元句有两种词序：pp.35-36

| 词序 | 结构 | 频率 |
|------|------|------|
| **A型** | V + 主事者(ko) + 受事者(to) | **较常见** |
| **B型** | V + 受事者(to) + 主事者(ko) | 可行 |

| 阿美语（A型） | 阿美语（B型） | 汉语 |
|-------------|-------------|------|
| Komaen **ci Mayaw** to fonga. | Komaen to fonga **ci Mayaw**. | Mayaw吃地瓜。 |
| Mi'atal **ko singsi** to hemay. | Mi'atal to hemay **ko singsi**. | 老师在盛饭。 |

[FACT] A型词序（主事者在前）"较常见"，但 B型也是可以接受的。p.36

[INFERENCE] B型词序可能用于特定语用目的（如强调受事者、引入新信息等），但教材未提供具体的语用分析。建议初学者先掌握 A 型词序，后续再学习 B 型的语境使用。

### 5. 当主语是代名词时

[FACT] 人称代名词（主格形式）直接替代 ko/ci + 名词的位置，不需要再加 ko 或 ci。pp.48-49

| 阿美语 | 分析 | 汉语 |
|--------|------|------|
| Komaen **kako** to fonga. | 吃·AF + 我(主格) + to+地瓜 | 我吃地瓜。 |
| Mifoting **kiso** to foting. | 捕鱼·AF + 你(主格) + to+鱼 | 你在捕鱼。 |
| Mi'atal **cingra** to hemay. | 盛·AF + 他(主格) + to+饭 | 他在盛饭。 |

回顾第3-4课学过的代名词：
- kako = 我（主格）
- kiso = 你（主格）
- cingra/cira = 他/她（主格）

### 6. 否定形式

[FACT] AF 及物句的否定：在句首加 **caay**，动词的焦点标记变化：pp.37, 93-97

| 肯定 | 否定 |
|------|------|
| K**\<om\>**aen ci Mayaw to fonga. | **Caay** **ka**-k\<om\>aen ci Mayaw to fonga. |
| **Mi**foting ci Mayaw. | **Caay** **pi**-foting ci Mayaw. |

**否定焦点变换规则**：

| 肯定标记 | 否定形式 |
|---------|---------|
| mi- → | **pi-** |
| ma- (AF) → | **ka-** |
| <om> → | **ka-**... |

[FACT] pp.93-97

> **注意**：否定句的焦点变换比较特殊，将在第16课详细讲解。本课先了解基本模式即可。

## 格位系统初步：三分的轮廓

至此，我们已经看到了阿美语普通名词格位系统的**全部三个标记**：

| 标记 | 格位 | AF句功能 | 例句 |
|------|------|---------|------|
| **ko** | 主格 (NOM) | 主事者（主语） | Komaen **ko** singsi to fonga. |
| **to** | 斜格 (OBL) | 受事者 | Komaen ci Mayaw **to** fonga. |
| **no** | 属格 (GEN) | 领属者（"的"） | fonga **no** singsi "老师的地瓜" |

[FACT] **no** 暂时只作为领属标记出现（第3课已见过），如 payso **no** singsi "老师的钱"。no 的另一重要功能——标记非 AF 句的主事者——将在第9课（焦点系统）学习。pp.43-44

```
        格位三分（普通名词）
        ┌──────────────────┐
        │  ko    主格 NOM  │ ← 标记主语（AF句的主事者）
        │  no    属格 GEN  │ ← 标记所有者 / 非AF句主事者
        │  to    斜格 OBL  │ ← 标记受事者 / 接受者 / 时间原因
        └──────────────────┘
```

## 语言对比 [COMPARISON]

| 特征 | 阿美语（AF句） | 汉语 | 英语 |
|------|-------------|------|------|
| 主语标记 | ko/ci（显性标记） | 无（靠语序） | 无（靠语序） |
| 宾语标记 | to（显性标记） | 无（靠语序） | 无（靠语序） |
| 语序灵活性 | 主事者和受事者可换位（因有标记） | 不可换位 | 不可换位（基本） |
| 动词一致 | 动词 AF 标记与主语 ko 呼应 | 无 | 主谓一致（人称/数） |

[FACT] 阿美语的格位标记使得词序比汉语更灵活。因为 ko 和 to 明确标示了谁是主语谁是宾语，所以"Mayaw ko fonga to"这样的歧义不会出现——你总能通过标记判断。pp.35-36
