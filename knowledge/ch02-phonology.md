**教材页码：** pp.05-18 | **提取日期：** 2026-05-19 | **状态：** 已确认

## 1. 章节概述

本章系统描述阿美语的书写符号系统、语音系统（辅音/元音的音位与同位音）和音韵系统（音节结构、重音、音韵规则），最后讨论现行书写系统的争议与拼写建议。

## 3. 核心知识条目

### 3.1 书写符号系统概况

| 属性 | 内容 |
|------|------|
| 功能 | 阿美语的罗马拼音正字法 |
| 颁布 | [FACT] 2005年12月15日由教育部及行政院原民会共同颁布 |
| 原则 | [FACT] 书写符号反映基本音位，略去发音细节 |
| 辅音字母 | [FACT] 18个：c d f h k l m n ng p r s t w x y ' ^ |
| 元音字母 | [FACT] 5个：a e i o u |
| 各方言分布 | [FACT] 所有字母各方言均有（表2.1, 2.2），o/u在北部分别称为u，其他方言混用 |
| 来源页码 | pp.05-06 |

### 3.2 辅音字母的分布限制

| 属性 | 内容 |
|------|------|
| 关于 x | [FACT] 辅音字母除 x 外均可出现于词首、词中、词尾。x 很少使用，不出现在词首和词中 |
| 关于 ^ | [FACT] 词首及词尾的喉塞音 ^ 常省略不记。^epah→epah, ina^→ina。本书多採省略不记的方式 |
| 关于 e | [FACT] 元音字母除 e 外均可出现于词首、词中、词尾。e (/ə/) 不出现在词尾 |
| 来源页码 | pp.07-08 |

### 3.3 辅音系统详解

#### 3.3.1 塞音（Stops）

| 属性 | 内容 |
|------|------|
| 音位 | [FACT] 5个清塞音 /p, t, k, ʡ, ʔ/，书写为 p, t, k, ', ^ |
| 特征 | [FACT] 皆为不送气清塞音。/p, t, k/ 接近中文ㄅ、ㄉ、ㄍ，区别于送气ㄆ、ㄊ、ㄎ |
| 咽头塞音 | [FACT] /ʡ/ 书写为 '，在中文中无对应音 |
| 喉塞音 | [FACT] /ʔ/ 书写为 ^，在中文中无对应音 |
| 分布 | [FACT] 五音分布自由，可出现在词首、词中、词尾，具音位地位 |
| 来源页码 | pp.10-11 |

#### 3.3.2 擦音（Fricatives）

| 属性 | 内容 |
|------|------|
| 音位 | [FACT] 4个清擦音 /f, s, h, x/ |
| /f/ 方言变异 | [FACT] 唇齿擦音，越北部越接近 [b]，越南部越接近 [f] |
| /s/ 顎化 | [FACT] [si] 音串不出现 → /s/ 在 /i/ 前变为舌面擦音 [ɕ]（类似中文ㄒ），如 siwa [ɕiwa] "九" |
| /h/ | [FACT] 喉头擦音 |
| /x/ | [FACT] 舌根擦音，很少使用 |
| 来源页码 | pp.10-11 |

#### 3.3.3 塞擦音（Affricate）

| 属性 | 内容 |
|------|------|
| 音位 | [FACT] 仅一个 /ts/，书写为 c，不送气，类似中文ㄘ但不送气 |
| 顎化 | [FACT] [tsi] 音串不出现 → /ts/ 在 /i/ 前变为舌面塞擦音 [tɕ]（类似中文ㄐ），如 cima [tɕima] "谁" |
| 来源页码 | p.11 |

#### 3.3.4 鼻音（Nasals）

| 属性 | 内容 |
|------|------|
| 音位 | [FACT] 3个 /m, n, ŋ/，书写为 m, n, ng |
| /ŋ/ 对应 | [FACT] 无对应注音符号，但出现在中文ㄤ [aŋ] 及ㄥ [əŋ] 的尾音中 |
| 来源页码 | p.11 |

#### 3.3.5 边音（Lateral）

| 属性 | 内容 |
|------|------|
| 音位 | [FACT] 1个清边音 /ɬ/，书写为 d，中文无对应音 |
| 发音 | [FACT] 声带不振动，气流从舌侧送出，念法近似擦音 |
| 方言变异 | [FACT] 越北部越接近 [d]，越南部越接近 [ɬ] |
| 来源页码 | pp.11-12 |

#### 3.3.6 流音（Liquids）

| 属性 | 内容 |
|------|------|
| 颤音 | [FACT] /r/ 舌尖颤音，书写为 r |
| 闪音 | [FACT] /ɾ/ 舌尖闪音（舌尖轻拍齿龈一次），书写为 l |
| 来源页码 | p.12 |

#### 3.3.7 滑音（Glides）

| 属性 | 内容 |
|------|------|
| 音位 | [FACT] [j, w]，书写为 y, w |
| 特征 | [FACT] 常与元音结合形成复元音 |
| 滑音加插 | [FACT] 元音 /i/ 后接其他元音时加插 [j]；/o, u/ 后接其他元音时加插 [w]。如 fafahi+-an → fafahian [fafahijan] "女人" |
| 来源页码 | p.12 |

### 3.4 元音系统

| 属性 | 内容 |
|------|------|
| 音位数量 | [FACT] 4个元音音位：/i, u/o, a, ə/，书写为 i, u/o, a, e |
| /i/ | [FACT] 读如中文ㄧ |
| /a/ | [FACT] 读如中文ㄚ |
| /ə/ (e) | [FACT] 发音与ㄜ近似 |
| /u/与/o/ 关系 | [FACT] 为自由变体（free variation），无辨义作用。发音近似 [ʊ] |
| /o/的条件变异 | [FACT] 当该元音后面有舌根擦音 x、咽头塞音 ' 及喉头擦音 h 且为同一音节时，发音接近ㄛ。如 capox, tono', kohting |
| [e]的来源 | [FACT] 近似中前元音 [e] 的语音是由 /i/ 演变而来（元音舌位降低规则） |
| 方言使用 | [FACT] 北部方言习惯用 u，其他方言 o/u 混用。日语借词多用 u（如 ucya "茶"） |
| 来源页码 | pp.12-14 |

### 3.6 音节结构

| 属性 | 内容 |
|------|------|
| 基本特征 | [FACT] 音节一般以辅音开始，声母辅音至多一个，允许至多以1个辅音结尾的闭音节 |
| 最大音节 | [FACT] CVVC（4个语音），其中VV为复元音韵核，实际发音近似VG或GV组合 |
| 音节类型 | [FACT] 7种：V, VC, CV, CVC, CVV(CGV), CVV(CVG), CVVC(CGVC) |
| 来源页码 | pp.15-16 |

### 3.7 重音规则

| 属性 | 内容 |
|------|------|
| 基本规则 | [FACT] 重音落在词的最后一个音节，故不特别标示 |
| 加缀后移动 | [FACT] 加词缀时重音随之后移，仍落在新词最后音节。如加 -an, -en 等后缀时重音落在后缀上 |
| 是非问句 | [FACT] 是非问句中，最后词的倒数第二音节语调升高（↗），最后音节下降至平调，重音移至倒数第二音节 |
| 强调语气 | [FACT] 为表强调，倒数第二音节元音延长，重音也落在该音节 |
| 叙述句语调 | [FACT] 下降调（↘） |
| 来源页码 | pp.16-17 |

### 3.9 书写系统争议与建议

| 属性 | 内容 |
|------|------|
| 咽头/喉塞音区分 | [FACT] 李壬癸(1992)建议：以 ' 表喉塞音/ʔ/，以 q 表咽头塞音/ʡ/。现行系统以 ' 表咽头塞音易造成跨语言混淆 |
| o/u 统一 | [FACT] o/u 虽为自由变体，但建议各方言择一符号使用，以利词汇统一拼写与保存 |
| 长元音符号 | [FACT] 元音拉长符号（:）不适合纳入系统，因为长元音非基本音位 |
| 大写规则 | [FACT] 专有名词及句首单词词首须大写，但 ' 和 ^ 之后字母不应大写（如 'amis 的 a 不大写），因为这两个符号也是字母 |
| 来源页码 | pp.17-18 |
