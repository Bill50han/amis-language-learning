# 第九章：使動結構

**教材页码：** pp.83-91 | **提取日期：** 2026-05-19 | **状态：** 已确认

## 1. 章节概述

使动结构（causative）指某一动作导致另一动作/事件/状态的发生。比非使动结构增加一个论元（使动者）。阿美语有三种形成方式：词汇式、构词式（pa-前缀为主）、句法式。

## 2. 核心知识条目

### 2.1 三种使动结构

| 类型 | 机制 | 特征 | 例句 |
|------|------|------|------|
| **词汇式** | 本身带使动语义的词 | 数量少，不须pa- | pacok "宰杀" (=使死), feli "给" (=使有) |
| **构词式** | pa- 前缀 | 最常见 | pa-kaen "喂" (=使吃), pa-rakat "开车" (=使走) |
| **句法式** | 使动动词 + 其他动词 | 复杂句，枢轴结构 | patangic "强迫/哀求" + V |

来源页码：pp.83-88

### 2.2 pa- 前缀的核心用法

| 词根类别 | pa- 语义 | 例子 | 来源 |
|---------|---------|------|------|
| 名词 | "使有、使增加" | dateng "菜" → pa-dateng "加菜" | p.87 |
| 形容词 | "使变成" | kaeped "软" → pa-kaeped "使变软" | p.87 |
| 动词 | "使做动作" | kaen "吃" → pa-kaen "喂" | p.87 |
| 交易动词 | 改变方向 | cakay "买" → pacakay "卖"; caliw "借入" → pacaliw "借出" | p.84 |

### 2.3 mi- 和 -en 的使动用法

| 属性 | 内容 |
|------|------|
| 规则 | [FACT] mi- 和 -en 加在静态动词词根上也可衍生使动语意，不须加 pa- |
| mi- 例 | mi-lasang "使醉" ← lasang "醉" |
| -en 例 | fa'edet-en "加热" ← fa'edet "热" |
| pa- vs mi- 对比 | [FACT] 同一词根可有不同构词方式，语意不同。如 keced "缩小"：mi-keced "使缩小" vs pa-keced "请人改小" |

来源页码：pp.85-86

### 2.4 pa-pi- 弱使动

| 属性 | 内容 |
|------|------|
| 对比 | [FACT] pa- = 直接使动；pa-pi- = 弱使动（"请/叫...做"） |
| pa-例 | Pa-rakat-en nira koni a paliding. "他开这辆车。(直接使车走)" |
| pa-pi-例 | Pa-pi-rakat-en nira ci Panay. "他请Panay去走路。" (*他使Panay走路) |

[FACT] pa-pi-使动句的主语为**使动者**，受动者带斜格/宾格标记。

来源页码：pp.87-88

### 2.5 句法式使动结构

| 属性 | 内容 |
|------|------|
| 结构 | [FACT] 使动动词(patangic等) + 使动者 + 受动者 + 其他动词 |
| 特征 | [FACT] 为复杂句（含至少两个动词），受动者同时是第二个动词的主事者→形成**枢轴结构** |
| 例 | Patangic kako ci Akian tayra i Taypak. "我强迫Aki去台北。" |

来源页码：p.88

### 2.6 使动结构的主语选择

#### 双论元使动句

| 焦点 | 主语(ko/ci) | 来源 |
|------|------------|------|
| AF | 使动者 | p.89 |
| PF | 受动者 | p.89 |

#### 三论元使动句（三种类型）

| 类型 | PF主语可选 | 例词 |
|------|----------|------|
| **仅受动者可作主语** | 受动者✓ 受役物✗ | pa-kaen "喂" (受动者=被喂者) |
| **仅受役物可作主语** | 受动者✗ 受役物✓ | pa-resap "使散播" (受役物=话) |
| **两者皆可作主语** | 受动者✓ 受役物✓ | pa-kalat "给...咬" |

来源页码：pp.89-91
