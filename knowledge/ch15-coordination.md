# 第十五章：並列結構

**教材页码：** pp.129-132 | **提取日期：** 2026-05-19 | **状态：** 已确认

## 1. 核心知识

### 1.1 三类并连结构

| 类型 | 连接成分 | 适用范畴 |
|------|---------|---------|
| 对等并列 | **ato, aci**（名词组）；无连接词（动词组/子句） | 名词组、动词组、子句 |
| 非对等并列 | **ano ca i** "要不然呢？" | 子句/简化子句（见Ch11选择问句） |
| 反义并列 | **nikaorira** "但是" 或直接并列 | 子句 |

### 1.2 名词组对等并列

| 连接词 | 适用对象 | 例句 |
|--------|---------|------|
| **ato** | 任何名词组（普通/专有/代名词） | foting **ato** hemay "鱼和饭"; ci mama **ato** ci ina "爸爸和妈妈" |
| **aci** | 仅人稱专有名词 | ci mama **aci** ina "爸爸和妈妈" |

#### 格位标记规则

| 规则 | 说明 |
|------|------|
| [FACT] 第二个连接成分前的格位标记**不出现** | ...ato hemay（不加 ko） |
| [FACT] 例外：属格标记和人称专有名词**斜格-an**必须保留 | Mitala kako ci Aki-**an** aci Panay-**an**. "*去掉-an不合语法" |
| [FACT] 三个名词组并列时，仅最后一个前出现连接词 | titi, 'ayam **ato** foting "猪肉、鸡和鱼" |

### 1.3 动词组/子句对等并列

| 特征 | 内容 |
|------|------|
| 标记 | [FACT] **不需要**任何连接词，直接并列 |
| 动词组例 | Romadiw ci Sawmah masakero. "Sawmah一边唱歌一边跳舞。" |
| 子句例 | Maala no wawa ko piyang, makaen to nira. "小孩拿了糖果，吃了。" |

### 1.4 反义并列

| 特征 | 内容 |
|------|------|
| 主要方式 | [FACT] **nikaorira** "但是" |
| 替代方式 | [FACT] 两个意思相反的子句可直接并列（无连接词） |
| 例 | ...nikaorira, mafokil kako to ngangan nira. "...但是我不知道他的名字。" |
