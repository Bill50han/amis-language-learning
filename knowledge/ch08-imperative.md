# 第八章：祈使句

**教材页码：** pp.79-82 | **提取日期：** 2026-05-19 | **状态：** 已确认

## 1. 章节概述

阿美语祈使句与焦点系统互动。分肯定/否定两类，否定用 `aka` "别"。肯定祈使句焦点形式有三种：主事焦点(pi-/ka-)、受事焦点(-en)、工具施用(sa-...-en)。另有一种用引述动词 `han` 的叮咛型祈使句。

## 2. 核心知识条目

### 2.1 祈使句焦点形式对照（表8.1）

| 焦点类型 | 肯定叙述句 | 祈使式 | 规劝式 |
|---------|-----------|--------|--------|
| 主事焦点 | mi-, ma-, <om> | **pi-**, **ka-** | 同叙述句 |
| 受事焦点 | ma-; -en | **ka-**; **-en** | |
| 工具施用 | sa- | **sa-...-en** | |
| 处所施用 | -an | — | |

[FACT] 规劝式祈使句与肯定叙述句相同，只用第一人称复数包含式 kita/ita，无特殊句型。

### 2.2 肯定祈使句

#### 主事焦点祈使

| 标记 | 对应叙述句 | 例句 | 语意 |
|------|----------|------|------|
| **pi-** | mi- 动词 | **Pi**na'on to likelon no lalan! | "要小心路况！" |
| **ka-** | ma- 动词 | **Ka**tayal i Taypak! | "在台北工作！" |
| **ka-** | <om> 动词 | **Ka**l\<om\>owad! | "起立！" |
| **ka-** | 方位 | **Ka**itini ko pafongohan a mafoti'! | "把头枕在这里睡觉！" |

[FACT] 主事者第二人称代名词大多省略。

#### 受事焦点祈使

| 标记 | 例句 | 语意 |
|------|------|------|
| **-en** | Asip**en** ko cecay a liyad! | "请朗读第一课！" |
| **-en** | Lolod**en** ko sikal! | "请把竹蓆捲收起来！" |

#### 工具施用祈使

| 标记 | 例句 | 语意 |
|------|------|------|
| **sa-...-en** | **Sa**piletek**en** to kilang kira cakelis! | "用那斧头砍树！" |

[FACT] 强调处所的祈使句不是用处所施用结构，而是用 `ka-i-` + 方位名词（属主事焦点）。

### 2.3 否定祈使句：aka

| 属性 | 内容 |
|------|------|
| 否定词 | [FACT] **aka** "别/不要"，置于祈使动词前 |
| 动词属性 | [FACT] aka 本身带有动词属性，可加时貌标记 =to |
| 例句 | Aka **pi**hinokop a minanom! "不要趴着喝水！" |
| | Aka=to kararom kiso! "你不要伤心了！" |
| | Aka tengil**en** kora sowal! "不要听信那个谣言！" |

### 2.4 han 型叮咛祈使句

| 属性 | 内容 |
|------|------|
| 形式 | [FACT] 动词词根 + **han**（引述动词，受事焦点） |
| 功能 | [FACT] 带叮咛语气，非马上执行的命令 |
| 否定 | [FACT] 否定词 aka 出现在 han 前（非句首） |
| 例句 | Cokcok **han** to ko loma'! "记得把家里锁住！" |
| 否定例 | 'enoc **aka han** ko 'okak no foting! "不要嚥下鱼骨！" |

## 3. 祈使句形式规则总结

```
肯定祈使：
  主事焦点: pi-/ka- + 词根
  受事焦点: ka- + 词根 / 词根-en
  工具施用: sa-pi-/ka- + 词根-en
  叮咛型:   词根 + han

否定祈使：
  aka + (肯定祈使句)
  aka + =to (可选) + 肯定祈使句
```
