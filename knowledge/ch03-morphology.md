# 第三章：詞彙與構詞

**教材页码：** pp.19-31 | **提取日期：** 2026-05-19 | **状态：** 已确认

## 1. 章节概述

本章系统描述阿美语的词彙结构，包括：主要构词单位（词/词素/词根/词幹/词缀/依附词）、三大构词方法（加缀、重叠、复合）、拟声词、借词、以及各类词类特征。

## 3. 核心知识条目

### 3.1 基本概念定义

#### 3.1.1 词的分类

| 分类维度 | 类别 | 说明 | 页码 |
|---------|------|------|------|
| 音节多寡 | 单音节词 | ko "主格", haw "吗" | p.19 |
| | 多音节词 | foting "鱼" (2), 'akaway "茎" (3) | p.19 |
| 组成成分 | 单纯词 | 由一个成分组成，不可再分解 | p.19 |
| | 复杂词 | 由2+成分组成：mi+foting "捕鱼" | p.19 |
| 语义句法 | 实词（开放类） | 具实质语义：名词、动词 | p.19 |
| | 虚词（封闭类） | 标示句法关系：格位标记、代名词、连接词、助词 | p.19 |
| 来源 | 本族词 vs 外来词 | 借自日、英、闽南、汉语 | p.19 |

#### 3.1.2 词素类型

| 类型 | 定义 | 符号 | 例子 | 页码 |
|------|------|------|------|------|
| 自由词素 | 可独立存在的词素 | 无标记 | foting "鱼", ko "主格" | p.20 |
| 附著词素-词缀 | 必须附加在词上，不能单独使用 | `-` 或 `< >` | pi-, -an, <om> | p.21 |
| 附著词素-依附词 | 必须附加但不能单独使用 | `=` | =ako "我.属格" | p.21 |

#### 3.1.3 词缀 vs 依附词 对比

| 特征 | 词缀 | 依附词 | 页码 |
|------|------|--------|------|
| 附著对象 | 仅限词，有语义及类别限制 | 不选寄主词类别，可附加在多种位置 | p.21 |
| 重音影响 | 改变原词重音位置 | 不改变原词重音 | p.21 |
| 堆叠 | 后缀只能一个；前缀可堆叠 | 可堆叠（如 =to=ako） | pp.21-22 |
| 例 | 'adop-en "打猎-PF" (重音移至e) | 'adopen=ako "我打猎" | p.21 |

#### 3.2.2 中缀（表3.2）

| 属性 | 内容 |
|------|------|
| 标记 | [FACT] 仅一个中缀 `<om>`，形成动词，语义与原词幹相关 |
| 例子 | r\<om\>adiw "唱歌" ← radiw "歌"; t\<om\>angic "哭" ← tangic; l\<om\>owad "起立/起床" ← lowad |
| 来源页码 | p.23 |

#### 3.2.3 后綴（表3.3）

| 后綴 | 功能 | 例子 | 页码 |
|------|------|------|------|
| -ay | 形成名词："…的（人或事）" | maca'or-ay "被赶上的（人）", mifoting-ay "渔夫" | p.23 |

#### 3.2.4 环缀（表3.4）

| 环缀 | 功能 | 例子 | 页码 |
|------|------|------|------|
| pi-...-an | 形成名词："与语义相关的地点" | pi-codad-an "学校" ← codad "书"; pi-^ap^ap-an "抓青蛙的地方" | p.24 |
| sa-...-an | 形成动词："想做某事" | sa-picoco-an "想喝奶"; sa-piocor-an "想派遣" | p.24 |

### 3.3 重叠（Reduplication）

#### 3.3.1 词汇化重叠

| 属性 | 内容 |
|------|------|
| 定义 | [FACT] 由两个相同音段组合而成的单词，但该音段在语言中已不具意义，不可单独存在 |
| 例子 | rohroh "雷声", dokdok "偷偷的/闭门造车", filfil "嘴唇", 'aw'aw "狗吠声", ^es^es "吹哨声", fafa "背（动词）" |
| 注意 | [FACT] *roh, *dok, *fil, *'aw, *^es, *fa 在阿美语中均不单独成词 |
| 来源页码 | pp.24-25 |

#### 3.3.2 双音节重叠（CVCV）

| 属性 | 内容 |
|------|------|
| 规则 | [FACT] 重叠词根的前两个CV音节形成CVCV重叠 |
| 名词重叠 | [FACT] 表示复数/数量变多：wawa-wawa "很多小孩", wida-widang "很多朋友" |
| 动态动词重叠 | [FACT] 表示动作持续/反复：raka-rakat "走来走去", foti-foti' "重复做睡的动作" |
| 静态动词重叠 | [FACT] 表示程度加强或分布性语义 |
| 与sa-连用 | [FACT] sa- + CVCV 重叠表"…的样子/假装"：sa-raka-rakat "走来走去的样子", sa-foti-foti' "装睡" |
| 来源页码 | p.25 |

#### 3.3.3 Ca重叠（表3.5）

| 属性 | 内容 |
|------|------|
| 规则 | [FACT] 重叠词幹第一个辅音 + 元音 a |
| 与数词连用 | [FACT] 表人的个数：ta-tosa "两个人", ^a-^enem "六个人", pa-pina "多少人" |
| 与动词连用 | [FACT] 表未实现（未来）：ma-micokor "将要支持", ka-k\<om\>aen "将要吃", ta-tolon-en "将要祷告（的内容）" |
| 交互结构 | [FACT] ma-Ca-/ka-Ca- + 词根表互相：ma-ca-coli "互相吵架", ma-^a-^iric "互相扭捏", ka-sa-sorar "要互相退让" |
| 来源页码 | pp.25-26 |

#### 3.3.4 特殊形式重叠

| 属性 | 内容 |
|------|------|
| 模式 | [FACT] 重叠C₁V₁.C₂V₂.C₃V₂(C₄)中C₂.C₃V₂部分插入第一CV后 |
| 例子 | tamdamdaw "人人" ← tamdaw "人" (ta-m.da-m.adw) |
| 历史例 | pawliwli "串串香蕉" ← pawli（Zeng 1991记载，现已少用） |
| 现状 | [FACT] 已很少使用，现多用Ca重疊（pa-pawli）或全部重疊（pawli-pawli）替代 |
| 来源页码 | p.26 |

### 3.4 复合（Compounding）

| 类型 | 连接方式 | 结构 | 例子 | 页码 |
|------|---------|------|------|------|
| a-连接 | 连繫词 a | N + a + N | fafahian a kaka "姐姐", kilang a kayakay "木桥" | p.26 |
| 属格连接 | no/ni | N + no/ni + N | 'alo no fo'is "银河" (河流+属格+星星), teker ni Idek "彩虹" | p.26 |

### 3.5 借词（Loanwords）（表3.7摘要）

| 来源语言 | 数量判断 | 典型例词 |
|---------|---------|---------|
| 日语 | [INFERENCE] 最多 | takosi "计程车", otofay "摩托车", singsi "老师", dingki "电灯/电器", ucya "茶", toki "时间" |
| 闽南语 | 次多 | ising "医生", piyang "饼", saytaw "白萝卜", ekim "金子" |
| 汉语 | 较少 | congtong "总统", kawcong "高中", kocong "国中", pinsiyang "冰箱" |
| 英语 | 极少 | kolismas "圣诞节(Christmas)" |

借词涵盖范围：交通工具、人物职业、学校场所、电器用品、饮食、水果蔬菜、器物、机械、服装、休闲、宗教信仰、动物、天然资源等。

### 3.6 词类类别概览

| 词类 | 开放性 | 核心特征 | 详见章节 | 页码 |
|------|--------|---------|---------|------|
| 名词 | 开放 | 无性别标记；复数可借重叠标示但不强制；分人称专有/普通名词；前有格位标记 | Ch5 | pp.29-30 |
| 动词 | 开放 | 最大特征为焦点词缀；静态动词表达修饰概念（大/小/新/旧等） | Ch6 | p.30 |
| 格位标记 | 封闭 | 出现在名词组前，标示语意角色/语法关系（主格/属格/斜格） | Ch5 | p.30 |
| 人称代名词 | 封闭 | 自由形（主/属/斜格）+ 附著形（仅属格）；第一人称复数分包含/排除 | Ch5 | p.30 |
| 指示代名词 | 封闭 | 4个，指近/远，可单独使用也可修饰名词 | Ch5 | p.30 |
| 疑问词 | 封闭 | 属名词/动词或其他词类 | Ch11 | p.30 |
| 惊叹词 | 封闭 | 表示感情或态度，通常出现在句首：acya "赞叹", anako' "疼痛" | p.31 |

## 4. 关键表格/范式

### 表3.1-3.4 总结：阿美语词缀系统一览

| 词缀类型 | 形式 | 功能 | 例子 |
|---------|------|------|------|
| 前缀 | mi- | 主事焦点 | mi-foting "捕鱼" |
| 前缀 | ci- | 拥有/生长/穿戴 | ci-payso "有钱" |
| 前缀 | hali- | 喜爱 | hali-^epah "爱喝酒" |
| 前缀 | pa- | 使动 | pa-rakat "开车" |
| 前缀 | sa- | 製作/当作/像/加强/工具 | sa-^epah "酿酒" |
| 前缀 | tala- | 前往 | tala-lotok "去山上" |
| 前缀 | paka- | 能够 | paka-'araw "能看到" |
| 前缀 | pi- | 名物化（行为） | pi-cocok "打针（行为）" |
| 前缀 | ka- | 语义相关名物化 | ka-etip "西边" |
| 中缀 | <om> | 形成动词（仅有的中缀） | r\<om\>adiw "唱歌" |
| 后缀 | -ay | 形成名词（…的人/事） | mifoting-ay "渔夫" |
| 后缀 | -an | 处所（常与pi-配） | pi-codad-an "学校" |
| 后缀 | -en | 受事焦点 | 'adop-en "(要)打猎" |
| 环缀 | pi-...-an | 处所名物化 | pi-codad-an "学校" |
| 环缀 | sa-...-an | 想做某事 | sa-picoco-an "想喝奶" |

### 构词分析示例（图3.1）

```

```
