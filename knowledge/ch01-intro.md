## 1. 章节概述

本章介绍台湾原住民族群概况、阿美族的分布与现状（人口、方言分类、自称）、以及全书的16章内容简介。本章不涉及具体语言结构分析。

### 3.2 台湾南岛语的地位

| 属性 | 内容 |
|------|------|
| 功能 | 说明台湾南岛语在南岛语系中的特殊地位 |
| 内容 | [FACT] 根据Blust(1991)，南岛语系约1,200种语言分10大支系，台湾南岛语涵盖其中9大支系，其余1,176种语言全部属于第10支系。这说明台湾是南岛语的发源地。 |
| 来源 | Nature杂志(2000), "Taiwan's Gift to the World" |
| 来源页码 | p.01 |

### 3.3 族称：Pangcah vs 'amis

| 属性 | 内容 |
|------|------|
| 功能 | 说明阿美族自称的来源和用法 |
| 自称 Pangcah | [FACT] 意为"人"、"种族"或"同族人"（吴明义 2013:附录19页） |
| 他称 'amis | [FACT] 意为"北方"，来源有二说：①台东卑南族对阿美族的称呼；②台东马兰阿美族人认为祖先来自北方，自称'amis |
| 使用趋势 | [FACT] 花莲地区倾向自称为Pangcah，台东地区倾向自称为'amis。受圣经翻译影响两种称呼已交互使用。Pangcah一词可扩及概称全台原住民。 |
| 来源页码 | pp.01-02（注脚1） |
| 国际语种代号 | [FACT] ISO 639-3: ami |

### 3.4 地理分布

| 属性 | 内容 |
|------|------|
| 位置 | [FACT] 中央山脉东侧，立雾溪以南，太平洋沿岸的东台纵谷及东海岸平原 |

| 特点 | [FACT] 大部分居住于平地，极少数居于山谷中 |
| 都市分布 | [FACT] 近年因就业因素移居，人口集中在都会区如新北市 |

### 3.6 方言分类

| 属性 | 内容 |
|------|------|
| 分类方式 | [FACT] 大致分为5-6群，存在多种分类体系（表1.1） |
| 主要分类 | [FACT] 目前最常采用土田滋教授的五分法：撒奇莱雅、北部方言、中部方言（太巴塱-马太安方言 + 秀姑峦）、南部方言（马兰、恒春） |
| 亲疏关系 | [FACT] 词彙统计法：撒奇莱雅和北部方言自成一支，其他各群另成一支；同源词分布：可分为北部、中部（秀姑峦、海岸）、南部（卑南、恒春） |
| 撒奇莱雅 | [FACT] 已于2007年获官方承认为台湾第13个原住民族，不再列为阿美语方言之一 |
| 本书依据 | [FACT] 本书描述秀姑峦阿美语（原民会分类），涵盖土田滋分类中的部分中部方言及太巴塱-马太安方言。中部方言是所有阿美语方言中使用最广者（Li 1997） |
| 来源页码 | pp.02-03 |
