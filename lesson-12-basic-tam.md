# 第12课：基本时貌——过去/完成/进行

## 本课目标

学完本课后，你将能够：
1. 理解阿美语 TAM（时貌语气）系统的四大手段
2. 使用 na- 表达过去事件
3. 使用 =to 和 =ho 表达完成和未完成
4. 根据焦点标记本身判断时间暗示
5. 将阿美语 TAM 标记与汉语"了/过/在"正确对应

---

## 核心知识

### 1. TAM 系统总览：四大手段

[FACT] 阿美语通过四种手段共同表达时貌语气（Tense-Aspect-Mood）：pp.68-73

| 手段 | 形式 | 功能 |
|------|------|------|
| **词缀** | na-（过去）、-ay（实现） | 附着在动词上，标示时间/实现状态 |
| **Ca-重叠** | Ca-重叠 + 动词 | 标示非实现（未来/从未发生） |
| **依附词** | =to（完成）、=ho（未完成） | 附着在句尾或动词后 |
| **焦点标记本身** | mi-/ma-/<om>/-en 等 | 焦点标记自身暗示时貌 |

```
动词 = (Ca-重叠) + (na-) + 焦点标记 + 词根 + (-ay) + (=ho/=to)
      [非实现]    [过去]   [AF/PF]           [实现]   [未完成/完成]
```

[COMPARISON] 汉语 TAM 系统主要靠**虚词**（了/过/在/将）和**语境**。阿美语更依赖**词缀**和**依附词**。英语用**助动词+屈折**（will eat / has eaten / is eating）。

### 2. 重要前提：焦点标记本身的时貌含义

[FACT] 在没有任何显性 TAM 标记的情况下，焦点标记本身暗示时貌：pp.69-71

| | 过去 | 非过去（进行） | 非过去（未来） |
|------|------|------|------|
| **AF** | — | mi-, <om>, ma- | mi- |
| **PF** | ma- | — | -en |

**这意味着**：

- AF 标记（mi-, <om>, ma-）在没有过去标记时，默认**非过去**——正在发生或将要发生
- PF ma- 默认**过去**——已经发生
- PF -en 默认**非过去**——将要/应该发生

**实例**：

| 阿美语 | 焦点 | 时间暗示 | 汉语 |
|--------|------|---------|------|
| Komaen ci Mayaw to fonga. | AF <om> | 非过去（正在/将要） | Mayaw在吃/要吃地瓜。 |
| **Ma**kaen ni Mayaw ko fonga. | PF ma- | 过去（已发生） | 地瓜被Mayaw吃了。 |
| **Kaen-en** ni Mayaw ko fonga. | PF -en | 非过去（将要） | 地瓜要被Mayaw吃了。 |

[FACT] 要表达确切的时间（过去/现在/未来），需要借助**时间副词**或显性 TAM 标记。pp.69-71

### 3. na- —— 过去标记

[FACT] **na-** 是前缀，加在焦点标记之前，标示**过去事件**或**相对过去**。pp.71

**构成**：na- + 焦点标记 + 词根

| 阿美语 | 结构 | 汉语 | 来源 |
|--------|------|------|------|
| **Na**-mi-kayakay to kilang kako. | na + mi-爬 + to+树 + 我 | 我曾经爬过树。 | p.71 |
| **Na**-ma-kaen ni Mayaw ko fonga. | na + ma-吃 + ni+Mayaw + ko+地瓜 | 地瓜已经被Mayaw吃了。（过去+完成） | — |
| **Na**-ma-tolo' ci Mayaw. | na + ma-跌倒 + ci+Mayaw | Mayaw跌倒了（在过去的某个时间）。 | — |

[COMPARISON] na- 的语义范围覆盖汉语的：
- "曾经"（经历体）：Namikayakay kako to kilang. "我**曾经**爬过树。"
- "了"（完成体，当强调过去时）：Namakaen ni Mayaw ko fonga. "地瓜被Mayaw吃**了**（在过去某时）。"

但 na- 更侧重**时间上的过去**，不像"了"还可以仅表完成（不涉及时间）。

### 4. =to —— 完成貌"了"

[FACT] **=to** 是依附词，标示**完成貌**——事件已经完成或状态已经变化。大致对应汉语的"了₂"（句尾"了"，表变化/完成）。pp.72-73

| 阿美语 | 结构分析 | 汉语 |
|--------|---------|------|
| Takaraw **=to** kina wawa. | 高 + =to(完成) + 这孩子 | 这孩子长高**了**。 |
| Ma'oning-ay **=to** ko fodoy. | 脏-实现 + =to(完成) + ko+衣服 | 衣服已经脏**了**。 |
| Makaen **=to** no singsi ko fonga. | PF.吃 + =to(完成) + no+老师 + ko+地瓜 | 地瓜已经被老师吃**了**。 |

[FACT] =to 可以出现在动词、谓语或句尾。它表示事件状态从"未完成"变为"完成"。pp.72-73

[COMPARISON] 阿美语的 =to 和汉语"了"高度对应：
- 汉语"了₁"（动词后，完成体）："吃了饭" → komaen=to to hemay
- 汉语"了₂"（句尾，变化体）："孩子高了" → takaraw=to ko wawa

=to 似乎同时覆盖"了₁"和"了₂"的功能。[INFERENCE] 需要更多教材例句来确认精确对应关系。

### 5. =ho —— 未完成貌"还（没）"

[FACT] **=ho** 是依附词，标示**未完成貌**或**持续貌**——事件还没完成，或状态仍在持续。对应汉语"还（没）"。pp.73

| 阿美语 | 汉语 |
|--------|------|
| Mangta'ay **=ho** kina minowad. | 这颗李子**还**很青涩。 |
| Caay **=ho** komaen ci Mayaw. | Mayaw**还没**吃。 |
| Mafoti'ay **=ho** ko wawa. | 孩子**还在**睡觉。 |

[FACT] =ho 的核心语义是"状态/事件尚未终结"。pp.73

[COMPARISON] =ho ≈ 汉语"还"——既可用于肯定（"还在"）也可用于否定（"还没"）。

### 6. 焦点-TAM 互动综合表

[FACT] 下表展示焦点标记和 TAM 标记如何配合：pp.69-71

```
┌──────────────────────────────────────────────────────┐
│              焦点-TAM 互动                            │
├──────────┬────────────┬────────────┬─────────────────┤
│          │  过去       │  非过去(进行) │  非过去(未来)    │
├──────────┼────────────┼────────────┼─────────────────┤
│ AF 标记   │ na- + V    │ mi-, <om>,  │ mi- (无显性标记)  │
│          │            │ ma-         │ 或 Ca-重叠+mi-   │
├──────────┼────────────┼────────────┼─────────────────┤
│ PF 标记   │ ma- 本身   │ —          │ -en (无显性标记)  │
│          │ 或 na-+ma- │            │ 或 Ca-重叠+ma-   │
└──────────┴────────────┴────────────┴─────────────────┘

再加: =to (完成), =ho (未完成), -ay (实现)
```

## 句型公式卡片

```
┌─────────────────────────────────────────────┐
│  TAM 标记速查                                │
│                                             │
│  na-     = 过去（"曾经/当时"）                │
│  =to     = 完成了（"了"）                     │
│  =ho     = 还没/还在（"还"）                  │
│                                             │
│  AF 默认  = 非过去（正在/将要）               │
│  PF ma-  = 过去（已发生）                    │
│  PF -en  = 非过去（将要/应该）                │
│                                             │
│  na- + ma-   = 过去+完成 = "已经...了"        │
│  =ho + caay  = 还没                           │
└─────────────────────────────────────────────┘
```

## 语言对比 [COMPARISON]

| 语义 | 阿美语 | 汉语 | 英语 |
|------|--------|------|------|
| 过去 | na- | "曾经/过" | past tense / used to |
| 完成 | =to | "了" | have done / perfect |

| 进行（非过去） | AF默认 / -ay | "在" | be V-ing |
| 将来 | -en默认 / Ca-重叠 | "将要" | will V |

### 2. 添加 TAM 标记

### 3. 翻译（汉→阿美）

| 汉语 | 阿美语 |
|------|--------|
| Mayaw已经吃了地瓜。 | |
| 这孩子长高了。 | |

### 4. 辨析

下列句子中的 =to 和 na- 可以互换吗？为什么？

- Takaraw **=to** kina wawa. "这孩子长高了。"
- **Na**mitolon to kilang kako. "我曾经爬过树。"

提示：=to 强调**完成/变化**（"从不高到高"），na- 强调**过去时间**（"在过去的某个时间"）。功能不同，不能互换。
