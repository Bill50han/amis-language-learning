# 第1課 阿美語的字母

> 对应单元：Unit 1（L01 字母总览 / L02 音节与重音）
> 语法重点：18辅音+5元音、重音最后音节、腭化规则
> 场景：Kacaw老师在部落教室教阿美语字母

---

## 基本課文

| 阿美語 | 中国語 |
|--------|--------|
| O singsi ci Kacaw. I pitilidan cingra. | Kacaw是老师。他在学校。 |
| "Anini, manam kita to tilid no 'Amis," saan ci Kacaw. | "今天，咱们学习阿美语的字母，"Kacaw说道。 |
| "Ira ko 18 a tatilidan no sowal no 'Amis," saan cingra. | "阿美语有18个辅音字母，"他说。 |
| "Ira ho ko 5 a tatilidan no ngiha'," saan cingra. | "还有5个元音字母，"他说。 |

| 阿美語 | 中国語 |
|--------|--------|
| "Tengilen ko ngiha' no matiniay a tilid: a, i, u, e." | "听这些字母的发音：a, i, u, e。" |
| "A — ira i mata, ina, ama." | "A — 在 mata(眼睛)、ina(妈妈)、ama(奶奶)里。" |
| "I — ira i cidal, lima, siri." | "I — 在 cidal(太阳)、lima(五)、siri(羊)里。" |
| "O ngiha' no c ato s, i 'ayaw no i, mafalic," saan ci Kacaw. | "c 和 s 的发音，在 i 前面，会变化，"Kacaw说道。 |
| "Cima — cima kiso? Cidal — i kakayayan ko cidal." | "Cima — 你是谁？Cidal — 太阳在天上。" |

| 阿美語 | 中国語 |
|--------|--------|
| "O sakakaay a ngiha' no sowal i, itiraay i saikoray a tilid," saan ci Kacaw. | "词语的重音，在最后一个音节，"Kacaw说。 |
| "Tengilen: i-NA, ma-TA, ta-MDAW, fa-fa-HI." | "听：i-NA, ma-TA, ta-MDAW, fa-fa-HI。" |
| "Tengil," saan ci Kacaw. "Roma-di-WIW." | "听，"Kacaw说。"Roma-di-WIW。" |
| "O ngiha' no ' ato ^, awaay i sowal no Congko," saan cingra. | "' 和 ^ 的音，在中文里没有，"他说。 |
| "'Ayam — 'ayam i kilang. Ma-fo-TI' — mafoti' ko wawa." | "鸟 — 鸟在树上。睡觉 — 孩子在睡觉。" |

---

## 文法筆記

1. **`O ... ci ...` 名詞述語句**（→L03）：`O singsi ci Kacaw.` = "Kacaw是老师。" O 標記謂語名詞，ci 標記主語（專有人名）。

2. **重音在最後音節**（→L02）：i-**NA**, ma-**TA**, tam-**DAW**, fa-fa-**HI**。加綴後重音隨之後移。

3. **腭化規則**（→L02）：c 和 s 在 i 前變為舌面音。`cima` → [tɕima]，`posi` → [puɕi]。

4. **`ira` 存在句**（→L14）：`Ira ko 18 a tatilidan.` = "有18個輔音字母。" ira + ko + 存在物。

5. **`saan` 引述動詞**（→L19）：`"...", saan ci Kacaw.` = "「…」，Kacaw說道。"

## 応用課文

| 阿美語 | 中国語 |
|--------|--------|
| I pitilidan, somowal ci Mayaw ato cafay nira. | 在學校，Mayaw和他的朋友在說話。 |
| "Cima ko singsi iso?" saan ko cafay. | "你的老師是誰？"朋友問。 |
| "Ci Kacaw ko singsi ako," saan ci Mayaw. | "Kacaw是我的老師，"Mayaw說。 |
| "Mataneng cingra haw?" saan ko cafay. | "他厲害嗎？"朋友問。 |
| "Mataneng cingra. Mafana' cingra to tilid no 'Amis," saan ci Mayaw. | "他很厲害。他懂阿美語的字母，"Mayaw說。 |
| "Mafana' kiso to tilid no 'Amis haw?" saan ko cafay. | "你懂阿美語字母嗎？"朋友問。 |
| "Mafana' kako. Manam kako to tilid no 'Amis i loma'," saan ci Mayaw. | "我懂。我在家裡學阿美語字母，"Mayaw說。 |
| "Makapahay a singsi ci Kacaw!" saan ko cafay. | "Kacaw真是個好老師！"朋友說。 |
