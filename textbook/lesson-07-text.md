# 第7課 聚會

> 对应单元：Unit 7（L18 使動結構 / L19 連動與補述 / L20 修飾結構 / L21 並列反身交互）
> 语法重点：pa-/pa-pi-使動、a連動、關係子句（主語限制）、ato/aci並列、nikaorira轉折、tireng/=to反身、ma-Ca-/mala-交互
> 场景：Mayaw、Panay、Aki、Kacaw四人週末聚會

## 文法筆記

1. **pa- 使動**（→L18）：`Pa-kaen-en ni Panay kita.`（Panay餵咱們）。pa- + kaen（吃）→ pakaen（餵，使吃）。`Pa-pi-cokcok-en`（叫…鎖門），pa-pi- 弱使動。

2. **連動結構 `a`**（→L19）：`Tayni kamo a malahok.`（你們來吃飯）。兩個動詞用 a 連接，共享 TAM/否定/祈使，表達單一事件。

3. **關係子句——主語限制**（→L20）：`o mialaay to codad ako a tamdaw`（拿走我書的那個人）。mialaay（拿走的）是 AF + -ay（實現），中心語"人"是關係子句中的主語。

4. **並列 `ato` / `nikaorira`**（→L21）：`ci Aki ato ci Kacaw`（Aki和Kacaw），ato 連接名詞組。`nikaorira`（但是）連接意思相反的兩個子句。

5. **交互 `ma-Ca-`**（→L21）：`macacoliay`（互相吵架的）← ma- + Ca- + coli。ma-Ca- + 詞根表達"互相做某事"。`malakapotay`（互為同學的）← mala- + kapot。

6. **反身 `tireng` / `=to`**（→L21）：`tireng nira`（他自己，名詞形式）、`kako=to`（我自己，附著式）。=to 與完成貌 =to 同形異義。

## 応用課文

| 阿美語 | 中国語 |
|--------|--------|
| "Tengilen kako a somowal to pakafana' no mako," saan ci Kacaw. | 「聽我說說我知道的事，」Kacaw說。 |
| "O cima ko mafana'ay a romadiw i tamowanan?" saan cingra. | 「你們中誰是會唱歌的人？」他說。 |
| "Ci Panay ko mafana'ay a romadiw. Mafana' cingra a romadiw to radiw no 'Amis," saan ci Mayaw. | 「Panay是會唱歌的人。她會唱阿美語的歌，」Mayaw說。 |
| "Romadiw kiso i 'ayaw no mita haw?" saan ci Aki. | 「你在咱們面前唱一首好嗎？」Aki說。 |
| "Romadiway kako, nikaorira, mafoti'ay=to kako anini," saan ci Panay. "Anodafak, romadiw kako i 'ayaw namo." | 「我會唱，但是，我今天已經睏了，」Panay說。「明天，我在你們面前唱。」 |

| 阿美語 | 中国語 |
|--------|--------|
| "Mafana'ay kako to cecay a radiw no 'Amis," saan ci Mayaw. | 「我知道一首阿美語的歌，」Mayaw說。 |
| "O maan ko ngangan no radiw?" saan ci Kacaw. | 「歌的名字是什麼？」Kacaw說。 |
| "'O ra^kes kona kilang,' saan ko radiw," saan ci Mayaw. | 「歌裡說：『這棵樹是樟樹』，」Mayaw說。 |
| "O mako a akong ko mitiliday to radiw kora," saan ci Mayaw. | 「寫那首歌的人是我外公，」Mayaw說。 |

| 阿美語 | 中国語 |
|--------|--------|
| "Ira=ho ko toki. Romadiw kita!" saan ci Aki. | 「還有時間。咱們唱歌吧！」Aki說。 |
| "Malahok=to kita. Romadiw=to kita," saan ci Panay. | 「咱們吃完飯了。咱們唱歌吧，」Panay說。 |
| Malingatay ko foral i kakayayan. Romadiway ko laloma' ni Mayaw ato widang nira. | 月亮掛在天上。Mayaw的家人和他的朋友們在唱歌。 |
| "O kapahay a dadaya koni. O fangcalay a romi'ad," saan ci Kacaw. | 「這是個美好的夜晚。美好的一天，」Kacaw說。 |
| "Aka katayal anodafak. Aka pikakot to toki. O 'orip no mita koni," saan ci Aki. | 「明天別工作了。別管時間了。這是咱們的生活，」Aki說。 |
