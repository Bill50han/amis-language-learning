# 第3課 家人

> 对应单元：Unit 3（L06 格位系統全景 / L07 人稱代名詞(上) / L08 人稱代名詞(下)與指示詞）
> 语法重点：ko/no/to/ci/ni/ca/na 格位系統、人稱代名詞主格/屬格/所有格、指示代名詞三距離
> 场景：Aki來Mayaw家作客，認識了Mayaw的家人

---

## 基本課文

| 阿美語 | 中国語 |
|--------|--------|
| Tayni ci Aki i loma' ni Mayaw. | Aki來到Mayaw的家。 |
| "O loma' no mako koni," saan ci Mayaw. | 「這是我家，」Mayaw說。 |
| "Tayra i loma' kita. Ma'araw iso ko laloma' ako," saan cingra. | 「咱們進屋裡去。你見見我的家人，」他說。 |

| 阿美語 | 中国語 |
|--------|--------|
| "O ina ako koni," saan ci Mayaw. | 「這是我媽媽，」Mayaw說。 |
| "O mama ako kora," saan cingra. | 「那是我爸爸，」他說。 |
| "Kapahay a ma'araw kiso, ina," saan ci Aki. | 「伯母你好，」Aki說。 |
| "Kapahay a ma'araw kiso," saan ci ina. "O widang ni Mayaw kiso?" | 「你好，」媽媽說。「你是Mayaw的朋友嗎？」 |
| "O widang nira kako. O ising kako," saan ci Aki. | 「我是他的朋友。我是醫生，」Aki說。 |

| 阿美語 | 中国語 |
|--------|--------|
| Somowal ci Mayaw: "O fafahi ni mama ko ina. O fa'inay ni ina ko mama." | Mayaw說：「媽媽是爸爸的太太。爸爸是媽媽的丈夫。」 |
| "O kaka ako ci Panay. O matayalay cingra i kingko," saan cingra. | 「我姐姐是Panay。她在銀行工作，」他說。 |
| "O safa ni Panay kako. O mitiliday kako i pitilidan," saan ci Mayaw. | 「我是Panay的弟弟。我在學校讀書，」Mayaw說。 |

| 阿美語 | 中国語 |
|--------|--------|
| "O maan kora a sasing i fawahan?" saan ci Aki. | 「門邊那張照片是什麼？」Aki問。 |
| "O sasing no laloma' ako kora. O mako a ina, o mako a mama, ci Panay, ato kako," saan ci Mayaw. | 「那是我家人的照片。我媽媽、我爸爸、Panay和我，」Mayaw說。 |
| "Ira ko cecay a wawa no miso haw?" saan ci Aki. | 「你有一個孩子嗎？」Aki問。 |
| "Caay. O safa ako ci Mayaw," saan ci Panay. | 「不是。Mayaw是我弟弟，」Panay說。 |
| "O wawa ni ina kami—ci Panay, kako, ato safa ako," saan ci Mayaw. | 「我們是媽媽的孩子——Panay、我、和我弟弟，」Mayaw說。 |

---

## 文法筆記

1. **屬格標記 `no` / `ni`**（→L06）：表領屬。"loma' **ni** Mayaw"（Mayaw的家）、"widang **no** mako"（我的朋友）。`no` 用於普通名詞，`ni` 用於專有人名單數。

2. **附著式屬格 `=ako`**（→L07）：`ina ako`（我的媽媽）、`safa ako`（我的弟弟。附著式直接粘在名詞後。

3. **所有格 `mako`**（→L08）：`o mako a ina`（我的媽媽，前置）。同理 `o miso a wawa`（你的孩子）。可與附著式互換：ina=ako = mako a ina。

4. **指示代名詞三距離**（→L08）：`koni`（近指，這）、`kora`（遠指可見，那）、`koya`（遠指不可見，那）。`O loma' no mako **koni**.`（這是我家）。

5. **包含式 vs 排除式**（→L07）：`kita`（咱們，含聽者）、`kami`（我們，排聽者）。「我們是媽媽的孩子」用 `kami`——這句話是對Aki說的，不包含Aki。

## 応用課文

| 阿美語 | 中国語 |
|--------|--------|
| "O cima kora a tamdaw i sasing?" saan ci Aki. | 「照片裡那個人是誰？」Aki問。 |
| "O mama no ina ako cingra. O akong ako cingra. Matayalay cingra i omah i 'ayaw," saan ci Mayaw. | 「他是我媽媽的爸爸。他是我外公。他以前在田裡工作，」Mayaw說。 |
| "O cima koni a tamdaw?" saan ci Aki. | 「這個人是誰？」Aki問。 |
| "O ama no ina ako koni. Tomeangicay kako i sasing," saan ci Mayaw. | 「這是我媽媽的媽媽（外婆）。照片裡我在哭，」Mayaw說。 |
| "O fangcalay a sasing koni. O kapot no laloma' iso," saan ci Aki. | 「這是一張好照片。你家人的合照，」Aki說。 |

| 阿美語 | 中国語 |
|--------|--------|
| "O cima ko matayalay i kingko?" saan ci ina. | 「誰在銀行工作？」媽媽問。 |
| "Ci Panay. O matayalay no kingko cingra," saan ci Mayaw. | 「Panay。她是銀行的職員，」Mayaw說。 |
| "O maan ko tayal ni Panay i kingko?" saan ci Aki. | 「Panay在銀行做什麼工作？」Aki問。 |
| "Mafana' cingra to payso no tamdaw," saan ci Mayaw. | 「她管別人的錢，」Mayaw說。 |
| "O tayal nira, o pafeli to payso i tamdaw," saan ci ina. | 「她的工作是給人們錢，」媽媽說。 |
| "Makapahay a tayal kora!" saan ci Aki. | 「那是份好工作！」Aki說。 |
