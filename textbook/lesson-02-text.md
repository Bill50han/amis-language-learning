# 第2課 日常

> 对应单元：Unit 2（L03 名詞述語句 / L04 不及物動詞 / L05 及物動詞 AF）
> 语法重点：O...ko...、mi-/ma-/<om>動詞、AF及物句 V+ko+主事者+to+受事者
> 场景：Panay下班回家，和弟弟Mayaw的日常對話

---

## 基本課文

| 阿美語 | 中国語 |
|--------|--------|
| O matayalay ci Panay. I kingko cingra a matayal. | Panay是上班族。她在銀行工作。 |
| O mitiliday ci Mayaw, safa nira. I pitilidan cingra. | Mayaw是學生，她的弟弟。他在學校。 |

| 阿美語 | 中国語 |
|--------|--------|
| Tayniay=to ci Panay i loma'. Macahiw cingra. | Panay已經回到家了。她肚子餓。 |
| "Macahiw kako," saan ci Panay. | 「我肚子餓，」Panay說。 |
| "Ira ko hemay i sefi," saan ci Mayaw. | 「廚房有飯，」Mayaw說。 |
| Mi'atal ci Panay to hemay. Komaen cingra. | Panay盛了飯。她吃飯。 |

| 阿美語 | 中国語 |
|--------|--------|
| Komaen ci Panay to hemay ato foting. | Panay吃著飯和魚。 |
| "O maan ko nikaenan iso anini?" saan ci Mayaw. | 「你今天吃了什麼？」Mayaw問。 |
| "Komaen kako to hemay ato foting. O fangcalay a komaen," saan ci Panay. | 「我吃了飯和魚。很好吃，」Panay說。 |
| Mafoti' ci Mayaw. "Mafoti'ay kako," saan cingra. | Mayaw睏了。「我睏了，」他說。 |
| "Mafoti'=ho kiso? Caay=ho ka-pi-foting kita anini a dadaya?" saan ci Panay. | 「你還要睡？咱們今晚不去捕魚嗎？」Panay說。 |

---

## 文法筆記

1. **`O ... ci ...` 與 `O ... ko ...`**（→L03）：名詞述語句。"O matayalay ci Panay."（Panay是上班族）。`O` 標記謂語、`ci` 標記主語（專有人名）。否定用 `Caay ko`。

2. **ma- 靜態動詞**（→L04）：`Macahiw kako.`（我肚子餓）。ma- 標記不由自主的狀態，主語是經歷者而非執行者。`Mafoti'`（睡覺）同理。

3. **mi- 動態動詞**（→L04-05）：`Mi'atal ci Panay to hemay.`（Panay盛飯）。mi- 標記主動、有意志的動作。及物時帶 `to + 受事者`。

4. **AF 及物句完整結構**（→L05）：`V.AF + ci/ko + 主事者 + to + 受事者`。如 `Komaen ci Panay to foting.`（Panay吃魚）。

5. **=to 與 =ho**（→L12）：`Tayniay=to`（已經來了/到家了），`=to` 完成貌。`Mafoti'=ho`（還在睡），`=ho` 未完成貌。

## 応用課文

| 阿美語 | 中国語 |
|--------|--------|
| I dadaya, tayni ci Aki, widang ni Mayaw. | 晚上，Mayaw的朋友Aki來了。 |
| "O ising ci Aki," saan ci Mayaw. | 「Aki是醫生，」Mayaw說。 |
| "O ising kiso? O matayalay kako," saan ci Panay. | 「你是醫生？我是上班族，」Panay說。 |
| "Macahiw kiso haw?" saan ci Panay. | 「你餓嗎？」Panay問。 |
| "Caay ka-cahiw kako. Komaenay=to kako to hemay i loma' no ising," saan ci Aki. | 「我不餓。我已經在醫院吃過飯了，」Aki說。 |

| 阿美語 | 中国語 |
|--------|--------|
| Mifoting kita anini a dadaya haw? | 咱們今晚去捕魚嗎？ |
| "Mifoting kita," saan ci Mayaw. | 「咱們去捕魚，」Mayaw說。 |
| "Mafoti'ay=ho kako..." saan ci Mayaw. | 「我還睏…」Mayaw說。 |
| "Aka=to pifoting anini!" saan ci Panay. "Mafoti'ay ko safa ako." | 「今晚別捕魚了！」Panay說。「我弟弟睏了。」 |
| "Anodafak, romakat kita i lotok," saan ci Aki. | 「明天，咱們去山上走走，」Aki說。 |
| "Makapahay! Romakat kita i lotok anodafak," saan ci Panay. | 「好！明天咱們去山上走走，」Panay說。 |
