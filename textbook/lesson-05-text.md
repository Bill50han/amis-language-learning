# 第5課 Kacaw老師的回憶

> 对应单元：Unit 5（L12 基本時貌 / L13 實現與非實現）
> 语法重点：na-過去、=to完成、=ho未完成、-ay實現、Ca-重疊非實現、TAM組合規則
> 场景：Kacaw老師的日記——回憶過去，談論現在，計劃未來

## 文法筆記

1. **na- 過去標記**（→L12）：`namitayal kako`（我工作了）、`namifoting kami`（我們捕了魚）。na- 加在焦點標記之前，標示過去事件。

2. **=to 完成貌**（→L12）：`Maherek=to.`（完成了）、`Namaherek=to.`（已經完成了）。=to 附著於動詞或句尾，表事件從"未完成"變為"完成"，≈漢語"了"。

3. **=ho 未完成貌**（→L12）：`mamitayal=ho kako`（我還要工作）。=ho 表事件尚未終結，≈漢語"還（沒）"。可與 caay 共現：`Caay=ho kaherek.`（還沒完成）。

4. **-ay 實現語氣**（→L13）：`mililocay kako`（正在洗澡）、`komaenay kako`（在吃飯）。-ay 後綴，標示事件正在發生或已發生，是現實的一部分。

5. **Ca-重疊 非實現**（→L13）：`mamitayal`（將要工作）、`Aira ko 'orad`（將有雨）。Ca-重疊取詞幹首輔音+a，加在焦點標記前，標示未來/從未發生的事件。

6. **TAM 組合**（→L13）：模板 `(Ca-) + (na-) + 焦點 + 詞根 + (-ay) + (=ho/=to)`。各標記可疊加使用。

## 応用課文

| 阿美語 | 中国語 |
|--------|--------|
| Somowal ci Panay to safa nira ci Mayawan: | Panay對她的弟弟Mayaw說： |
| "Na maherek no tayal, namacahiw kako," saan cingra. | 「工作完之後，我餓了，」她說。 |
| "Namikaen=to kiso to maan?" saan ci Mayaw. | 「你已經吃了什麼？」Mayaw問。 |
| "Namikaen kako to foting ato dateng," saan ci Panay. | 「我吃了魚和菜，」Panay說。 |
| "Caay=ho pikaen to hemay?" saan ci Mayaw. | 「還沒吃飯嗎？」Mayaw問。 |
| "Caay=ho. Makaen=to no mako ko foting, nikaorira, caay=ho pikaen to hemay," saan ci Panay. | 「還沒。魚已經被我吃了，但是，還沒吃飯，」Panay說。 |

| 阿美語 | 中国語 |
|--------|--------|
| "Oya mami' i cokoy, mamaheteng iso haw?" saan ci Mayaw. | 「桌上的那個橘子，你要吃掉嗎？」Mayaw問。 |
| "Mamaheteng no mako anodafak. Caay=ho kaheteng anini," saan ci Panay. | 「我明天吃掉它。今天還不（該）吃，」Panay說。 |
| "Mangta'ay=ho kora a mami' haw?" saan ci Mayaw. | 「那橘子還很青澀嗎？」Mayaw問。 |
| "Caay. Mahetengay=to kora a mami'. Ma'oningay=to, nikaorira, mafoti'ay=ho kako anini," saan ci Panay. | 「不。那橘子已經熟了。已經不青澀了，但是，我今天還要睡，」Panay說。 |
| "Anodafak, mamaheteng no miso kora a mami'," saan ci Mayaw. | 「明天，那橘子會被你吃掉，」Mayaw說。 |
| "Mamaheteng no mako anodafak," saan ci Panay. | 「我明天會吃掉它，」Panay說。 |
