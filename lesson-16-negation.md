# 第16课：否定句——"不是""没有""别"

## 本课目标

学完本课后，你将能够：
1. 正确选择三个否定词：caay/cai（叙述句）、aka（祈使句）、awa（存在/方位句）
2. 掌握否定句中的焦点变换规则（mi→pi, ma→ka 等）
3. 区分肯定句和否定句的动词形式
4. 理解否定焦点变换的完整范式

---

## 核心知识

### 1. 三个否定词的分工

[FACT] 阿美语有三个否定词系统，按句子类型严格分工：pp.93

| 否定词 | 否定范围 | 语义 | 例 |
|--------|---------|------|-----|
| **caay / cai** | 叙述句（动词/名词谓语） | "不/不是/没有" | Caay komaen ci Mayaw. "Mayaw没有吃。" |
| **aka** | 祈使句 | "别/不要" | Aka kaenen! "别吃！" |
| **awa** | 存在/所有/方位句（ira型） | "没有/不在" | Awa ko payso. "没有钱。" |

[FACT] 注意：i + 方位名词的方位句是介系词谓语属叙述句，用 **caay/cai** 否定（不是 awa）。pp.77, 93

### 2. 名词谓语句的否定

[FACT] 名词谓语否定：将 O 替换为 **caay ko**（或 cai ko）：pp.34, 93-94

| 肯定 | 否定 | 汉语 |
|------|------|------|
| **O** singsi ci Mayaw. | **Caay ko** singsi ci Mayaw. | Mayaw不是老师。 |
| **O** tamdaw kako. | **Caay ko** tamdaw kako. | 我不是人。 |

[FACT] caay + ka-O → caay kao（kao 或简化为 ko）。p.93-94

### 3. 动词谓语句的否定——焦点变换

[FACT] 否定句中，动词的焦点标记发生**系统性变换**——这正是第15课祈使句变换的否定版本：pp.94-97

| 焦点类型 | 肯定形式 | 否定形式 |
|---------|---------|---------|
| AF (mi-) | mi- + 词根 | caay **pi-** + 词根 |
| AF (ma-) | ma- + 词根 | caay **ka-** + 词根 |
| AF (<om>) | <om> + 词根 | caay **ka-<om>** + 词根 |
| PF (ma-) | ma- + 词根 | caay **ka-** + 词根 |
| PF (-en) | -en | caay **(ka-)**...**-en** |
| PF (sa- 工具施用) | sa- | caay **ka-sa-** |
| PF (-an 处所施用) | -an | caay **ka-...-an** |

[FACT] **核心规则**：mi → pi, ma → ka, <om> → ka-<om>（保留中缀）。pp.94-97

这与第15课祈使句的变换规则**几乎完全相同**。祈使和否定共享同一套焦点变换逻辑。

#### 例句对比

| 肯定 | 否定 | 汉语 |
|------|------|------|
| **Mi**foting ci Mayaw. | Caay **pi**foting ci Mayaw. | Mayaw没有捕鱼。 |
| **Ma**cahiw kako. | Caay **ka**cahiw kako. | 我不饿。 |
| T**\<om\>**angic kira wawa. | Caay **ka**-t**\<om\>**angic kira wawa. | 那小孩没有哭。 |
| **Ma**kaen no singsi ko fonga. | Caay **ka**-kaen no singsi ko fonga. | 地瓜没被老师吃。 |
| Kaen**en** no singsi ko fonga. | Caay **ka**-kaen**en** no singsi ko fonga. | 地瓜不会被老师吃。 |

[FACT] caay 和 cai 是变体关系，功能相同。caay 更常用。pp.93-94

### 4. 肯定 ↔ 否定 对照速查表

```
┌──────────────────────────────────────────────┐
│  肯定 → 否定 (叙述句)                          │
│                                              │
│  名词谓语:  O → caay ko                      │
│  mi-动词:   mi- → caay pi-                   │
│  ma-动词:   ma- → caay ka-                   │
│  <om>动词:  <om> → caay ka-<om>              │
│  PF ma-:    ma- → caay ka-                   │
│  PF -en:    -en → caay (ka-)...-en           │
│  sa-:       sa- → caay ka-sa-                │
│  -an:       -an → caay ka-...-an             │
│                                              │
│  祈使否定:  aka + V.祈使                       │
│  存在否定:  awa (替代 ira)                     │
└──────────────────────────────────────────────┘
```

### 5. 综合对比：三个否定词的使用场景

| 你想表达... | 用哪个否定词 | 例 |
|-----------|------------|-----|
| "X不是Y"（名词谓语否定） | **caay** | Caay ko singsi ci Mayaw. |
| "X没有做Y"（动词谓语否定 AF） | **caay** + pi-/ka- | Caay pifoting ci Mayaw. |
| "Y没有被做"（动词谓语否定 PF） | **caay** + ka- | Caay kakaen no singsi ko fonga. |
| "别做X！"（祈使否定） | **aka** | Aka pifoting! |
| "没有X / X不在"（存在/方位否定） | **awa** | Awa ko payso. |
| "不在L地方"（i型方位否定） | **caay** | Caay kai loma' cingra. |
