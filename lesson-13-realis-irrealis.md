# 第13课：实现与非实现—— -ay 与 Ca-重叠

## 本课目标

学完本课后，你将能够：
1. 理解"实现"（realis）和"非实现"（irrealis）的核心概念
2. 使用 -ay 后缀表达已实现/正在进行的事件
3. 使用 Ca-重叠表达未来/从未发生的事件
4. 掌握 TAM 标记的完整组合规则
5. 理解 TAM 系统如何与焦点系统互动

---

## 核心知识

### 1. 实现 vs 非实现 —— 阿美语的另一个维度

[FACT] 除了"过去/非过去"的时间维度，阿美语还有一个**实现性（realis/irrealis）**维度：pp.68-69

```
实现 (Realis)          非实现 (Irrealis)
├── 已经发生的           ├── 还没发生的
├── 正在发生的           ├── 从未发生的
└── 确定的事实           └── 假设、意愿、未来
```

[FACT] **实现** = 事件已经在现实中发生（或正在发生）；**非实现** = 事件还未在现实中发生。p.68

[COMPARISON] 汉语没有"实现/非实现"这个语法范畴。最接近的概念是"了/在"（实现）vs"将要/还没"（非实现），但汉语不强制标记这个区分。英语通过时态间接表达（past/present=realis，future/modal=irrealis）。

### 2. -ay 后缀 —— 实现语气

[FACT] **-ay** 是后缀，标示**实现语气**——事件正在发生或已经发生，是现实的一部分。pp.71-72

**构成**：焦点标记 + 词根 + **-ay**

| 阿美语 | 结构 | 汉语 | 来源 |
|--------|------|------|------|
| Mili**loc-ay** kako. | mi-洗澡 + -ay + 我 | 我正在洗澡。 | p.72 |
| Ma'on**ing-ay** ko fodoy. | ma-脏 + -ay + ko+衣服 | 衣服很脏。 | p.72 |
| Mafoti'**ay** ko wawa. | ma-睡觉 + -ay + ko+孩子 | 孩子在睡觉。 | — |

**-ay 的使用场景**：

| 场景 | 例句 | 含义 |
|------|------|------|
| 进行中的动作 | Mililoc-ay kako. | 我正在洗澡。 |
| 已存在的状态 | Ma'oning-ay ko fodoy. | 衣服（处于）脏（的状态）。 |
| 过去已发生的 | Na-mi-kayakay-**ay** kako. | 我曾经爬过树（且这是确定的事实）。 |

[FACT] -ay 可以与 na-（过去）和 =to（完成）共现：pp.71-72

| 阿美语 | TAM组合 | 汉语 |
|--------|---------|------|
| Na-mililoc-**ay** kako. | na- + -ay | 我（过去）在洗澡。 |
| Ma'oning-**ay** **=to** ko fodoy. | -ay + =to | 衣服已经脏了。 |

[COMPARISON] -ay 大致对应汉语"在"（进行体）和"着/了"（状态持续/实现）的语义范围，但更侧重于**事件是现实发生的**，而不只是时间位置。

### 3. Ca-重叠 —— 非实现语气

[FACT] **Ca-重叠**是动词词干（焦点标记+词根）第一个辅音+元音 a 的重叠，标示**非实现**——未来要发生的、或从未发生过的。pp.72

**构成**：Ca-重叠 + 焦点标记 + 词根

Ca-重叠的规则：取**动词词干（焦点标记+词根）**的第一个辅音 + a，将 Ca-前缀加在焦点标记之前。注意：取的是词干的首辅音，不是词根的首辅音。所以 mi-dotoc 取 m → **Ma**-mi-dotoc，而不取词根 d → *da-。

| 动词词干 | 词干首辅音 | Ca-重叠 | 完整形式 | 汉语 |
|---------|----------|---------|---------|------|
| **m**i-dotoc | m | **Ma**- | **Ma**-mi-dotoc ci Kacaw... | Kacaw将要继承... |
| **m**a-ngitid | m | **Ma**- | **Ma**-ma-ngitid no waco... | 将要被狗叼走... |
| **k**\<om\>aen | k | **Ka**- | **Ka**-komaen ci Mayaw... | Mayaw将要吃... |

**结构分析**：

```
Ma - mi - dotoc  ci Kacaw
│    │     │
Ca   焦点   词根
重叠  标记
(非实现)

→ "Kacaw将要继承..."
```

[FACT] Ca-重叠既可用于 AF 也可用于 PF：pp.72

| 焦点 | 例 |
|------|-----|
| AF 非实现 | **Ma**-mi-dotoc ci Kacaw... |
| PF 非实现 | **Ma**-ma-ngitid no waco... |

注意：Ca-重叠的 ma-（非实现标记）和 PF 的 ma- 不是同一个东西。前一个是 Ca 重叠形式，后一个是焦点标记。两者可以共现：Ma（非实现）+ ma（PF）+ ngitid。

[FACT] Ca-重叠的另一个功能是标示**从未发生**的事（非实现经历）：pp.72

| 阿美语 | 汉语 |
|--------|------|
| Caay ka-**ka**-komaen ci Mayaw to foting. | Mayaw从未吃过鱼。 |

### 4. TAM 完整组合规则

[FACT] 阿美语动词的 TAM 标记位置（模板）：pp.68-72

```
┌──────────────────────────────────────────────────────┐
│  (Ca-重叠) + (na-) + 焦点标记 + 词根 + (-ay) + (=ho/=to)  │
│  [非实现]    [过去]   [AF/PF]          [实现]   [完成/未完成] │
└──────────────────────────────────────────────────────┘
```

**常见组合**：

| 组合 | 功能 | 例 |
|------|------|-----|
| 焦点标记 + ∅ | AF=非过去进行；PF ma-=过去；PF -en=非过去 | Komaen ci Mayaw. "Mayaw在吃。" |
| na- + 焦点标记 | 过去 | Namifoting ci Mayaw. "Mayaw捕过鱼。" |
| 焦点标记 + -ay | 实现（正在进行/已实现） | Mililocay kako. "我在洗澡。" |
| 焦点标记 + =to | 完成 | Makaen=to. "被吃掉了。" |
| 焦点标记 + =ho | 未完成 | Mafoti'ay=ho. "还在睡。" |
| Ca-重叠 + 焦点标记 | 非实现（未来/从未） | Mamifoting ci Mayaw. "Mayaw将要捕鱼。" |
| na- + 焦点标记 + -ay | 过去+实现 | Namililocay kako. "我（当时）在洗澡。" |

### 5. Ca-重叠在关系子句中的应用（预告）

[FACT] Ca-重叠在关系子句（第14课）中也有重要功能——标示非实现的关系子句。pp.72, 119-127

| 关系子句时貌 | AF标记方式 | PF标记方式 |
|-----------|----------|----------|
| 实现 | mi-/ma-...-ay | ma-...-ay |
| 非实现 | **Ca-重叠**+mi-/ma- | **Ca-重叠**+ma-/-en |

这个用法将在第20课详细学习。

## TAM 系统完整公式卡片

```
┌──────────────────────────────────────────────────────┐
│             阿美语 TAM 系统                           │
│                                                      │
│  时制：                                               │
│    na-  = 过去                                       │
│    ∅   = 非过去（默认）                               │
│                                                      │
│  实现性：                                             │
│    -ay   = 实现（确定发生中/已发生）                    │
│    Ca-重叠 = 非实现（未来/从未发生/意愿）               │
│                                                      │
│  体貌：                                               │
│    =to   = 完成（"了"）                               │
│    =ho   = 未完成（"还"）                             │
│                                                      │
│  焦点自带：                                           │
│    AF      = 非过去（默认）                            │
│    PF ma-  = 过去（默认）                             │
│    PF -en  = 非过去（默认）                            │
│                                                      │
│  组合模板：                                           │
│    (Ca-) + (na-) + FOCUS + ROOT + (-ay) + (=ho/=to)  │
└──────────────────────────────────────────────────────┘
```

## 语言对比 [COMPARISON]

| 范畴 | 阿美语 | 汉语 | 英语 |
|------|--------|------|------|
| 实现/非实现 | **语法化**（-ay / Ca-重叠） | 未语法化 | 部分（will/would） |
| Ca-重叠 | 前缀性重叠，标示非实现 | 无对应 | 无对应 |
| -ay | 后缀，标示实现 | ≈ "在/了/着"的组合 | ≈ be V-ing / have done |
| TAM组合 | 多标记可共现 | 虚词可共现（"已经在做了"） | 助动词+屈折 |
