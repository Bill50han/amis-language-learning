# 第8课：人称代名词（下）与指示代名词

## 本课目标

学完本课后，你将能够：
1. 区分属格（=ako）和所有格（mako）的用法差异
2. 使用斜格人称代名词（takowanan / tisowanan 等）
3. 使用指示代名词表达"这个/那个"（三种距离 × 三种格位）
4. 使用地点指示代名词表达"这里/那里"

---

## 核心知识

### 1. 所有格代名词（mako / miso / mira...）——"的"的自由式

[FACT] 除了第7课学的附著式属格（=ako 粘在名词后），还有**所有格自由式**——可以独立使用，有自己独立的句子位置。pp.49-52

| 数/人称 | 所有格 | 汉语 |
|---------|--------|------|
| 单一 | **mako** | 我的 |
| 单二 | **miso** | 你的 |
| 单三 | **mira** | 他的/她的 |
| 复一含 | **mita** | 咱们的 |
| 复一排 | **niyam** | 我们的(排) |
| 复二 | **namo** | 你们的 |
| 复三 | **mira** | 他们的 |

#### 所有格 vs 附著式属格——关键对比

[FACT] 两者都可以表示"的"，但语法位置不同：pp.49-51

| | 附著式属格（=ako） | 所有格（mako） |
|------|-----------------|---------------|
| 独立性 | ❌ 必须依附名词后 | ✅ 可独立 |
| 位置1 | N + **=ako** | **mako** + a + N |
| 位置2 | — | N + no + **mako** |
| 与格标记共现 | ❌ 不能 | ✅ 必须（ko/no） |

**三种表达"我的孩子"的方式**：

| 方式 | 阿美语 | 结构 | 直译 |
|------|--------|------|------|
| 附著式 | wawa **=ako** | N + =ako | 孩子-我的 |
| 所有格·前置 | **mako** a wawa | 所有格 + a + N | 我的 的 孩子 |
| 所有格·后置 | wawa no **mako** | N + no + 所有格 | 孩子 的 我的 |

[FACT] 三种方式语义基本相同。"wawa=ako"最短、最常用。后两种方式用所有格 mako，但分别前置（+a）和后置（+no）。pp.49-51

[COMPARISON] 
- 附著式 =ako ≈ 汉语"我的"可直接说的简式（"我爸""我妈"中的"我"）
- 所有格前置 mako a ≈ 汉语正式"我的孩子"（名词短语完整形式）
- 所有格后置 no mako ≈ 类似日语"私の子供"的结构

#### 所有格的格位变化

[FACT] 所有格 mako/miso 等通过格位标记（ko/no）来表达不同的格位功能：p.49-51

| 功能 | 形式 | 例句 |
|------|------|------|
| 做主语 | **ko** mako | O tamdaw **ko mako** a wawa. "我的孩子是人。" |
| 做属格 | **no** mako | payso **no mako** a mama "我的爸爸的钱" |
| 做斜格 | **to** mako | — |

### 2. 斜格人称代名词——"对我/给你/在他那里"

[FACT] 所有人称代名词的斜格形式均以 **-an** 结尾，这是重要的形式标记。pp.52

| 数/人称 | 斜格 | 汉语 | 变体说明 |
|---------|------|------|---------|
| 单一 | **takowanan** / takonan | 对我/在我这里 | 两形并存 |
| 单二 | **tisowanan** / kisonan | 对你/在你那里 | 两形并存 |
| 单三 | **cingraan** / ciraan | 对他/在他那里 | — |
| 复一含 | **kitanan** / titaanan | 对咱们 | 两形并存 |
| 复一排 | **tamiyanan** / kaminan | 对我们(排) | 两形并存 |
| 复二 | **tamoanan** | 对你们 | — |
| 复三 | **cangraan** / cairaan | 对他们 | — |

[FACT] 变体形式可能反映方言差异和个人使用习惯，教材未给出系统分布规则。p.52

**例句**：

| 阿美语 | 汉语 | 来源 |
|--------|------|------|
| Pafeli ci Kolas to fonga **ci inaan** nira. | Kolas送地瓜**给他妈妈**。 | p.45 |
| Maolah kako **i tisowanan**. | 我爱你。（直译：我爱**于你**） | — |

[FACT] 斜格代名词常见功能：接受者（"给某人"）、处所目标（"到某人那里"）、被动经历者。pp.44-47

### 3. 指示代名词——"这个/那个"

[FACT] 阿美语指示代名词区分**三种距离**，每种有**三个格位**形式。其形式包含了格位标记成分（ko-/no-/to-）。pp.53-57

#### 标示人/事/物

| 距离 | 主格（ko-） | 属格（no-） | 斜格（to-） | 汉语 |
|------|-----------|-----------|-----------|------|
| **近** | **koni** | noni | toni | 这 |
| **远+可见** | **kora** / koraw | nora / noraw | tora / toraw | 那 |
| **远+不可见** | **koya** | noya | toya | 那 |

**记忆技巧**：
- 结尾 -ni → 近（koni ← ko + ni，"这"）
- 结尾 -ra → 远+可见（kora，"那个，看得见的"）
- 结尾 -ya → 远+不可见（koya，"那个，看不见的/前面提过的"）

[COMPARISON] 汉语"这/那"只区分两种距离。阿美语远指再分**可见/不可见**——这是汉语没有的维度。类似英语 this/that 也只分两种距离。可见/不可见的区分在南岛语中较常见。

#### 标示地点

| 距离 | 主格（ki-） | 属格（ni-） | 斜格（ti-） | 汉语 |
|------|-----------|-----------|-----------|------|
| **近** | **kini** | ninian | **tini** | 这里 |
| **远+可见** | **kira** / kiraw | nira / niraw | **tira** / tiraw | 那里 |
| **远+不可见** | **kiya** | niya | **tiya** | 那里 |

注意：地点指示词的前缀是 ki-/ni-/ti-（不是 ko-/no-/to-）。

### 4. 指示代名词的用法

#### 用法1：独立使用——代替整个名词组

| 阿美语 | 汉语 | 来源 |
|--------|------|------|
| O tamdaw **koni**. | **这**是人。 | — |
| O loma' **kora**. | **那**是房子。 | — |
| So'oten **koni** i tartar! | 把**这个**捆在柱子上！ | p.54 |
| Maolah kako to **koni**. | 我喜欢**这个**。 | — |

#### 用法2：修饰名词——"这个X"

[FACT] 指示词 + (a) + 名词。a 可省略，顺序**不可改变**（指示词必须在名词前）。pp.54-55

| 阿美语 | 汉语 |
|--------|------|
| **koni** a wawa | 这个孩子 |
| **kora** a foting | 那条鱼 |
| **koya** a loma' | 那间房子（不可见的/前面提过的） |

#### 用法3：融合形式 koni + a → kona

[FACT] koni + a 经常融合为 **kona** (/na/)：pp.54-55

```
koni + a → kona
```

| 阿美语 | 汉语 |
|--------|------|
| O ra^kes **kona** kilang. | 这棵树是樟树。 |
| **Kona** wawa, matanengay. | 这孩子啊，很聪明。 |

#### 用法4：地点指示词 + i

[FACT] 斜格地点指示词与 i "在"共现时标示地点：pp.56

| 阿美语 | 汉语 |
|--------|------|
| i **tini** | 在这里 |
| i **tira** | 在那里（可见） |
| i **tiya** | 在那里（不可见/从前） |

[FACT] **特殊用法**：i tiya ho = "从前"（字面："还在那时"）。p.57

## 句型公式卡片

```
┌─────────────────────────────────────────────┐
│  所有格（mako）的两种用法                     │
│    mako a N     = 我的X（前置，用 a 连接）     │
│    N no mako    = 我的X（后置，用 no 连接）    │
│                                             │
│  指示代名词                                  │
│    人/物：koni/kora/koya（主格）              │
│    地点：kini/kira/kiya（主格）               │
│    近/远(可见)/远(不可见) ← 三种距离          │
│    koni + a → kona（融合）                    │
│    地点用 i + 斜格：i tini "在这里"           │
└─────────────────────────────────────────────┘
```

## 例句

| 阿美语 | 汉语 | 来源 |
|--------|------|------|
| O tamdaw kako. | 我是人。 | — |
| Wawa=ako kora. | 那是我的孩子。 | — |
| **Mako** a wawa kora. | 那是**我的**孩子。 | — |
| O singsi ko **mako** a mama. | 我的爸爸是老师。 | — |
| O maan **koni**? | 这是什么？ | — |
| So'oten **koni** i tartar! | 把这个捆在柱子上！ | p.54 |
| O ra^kes **kona** kilang. | 这棵树是樟树。 | p.55 |
| I **tini** kako. | 我在这里。 | — |
| I **tira** ko loma' nira. | 他的房子在那边。 | — |
| I **tiya** ho... | 从前… | p.57 |
| Pafeli kako to fonga **i tisowanan**. | 我给你地瓜。 | — |
| Maolah kako **i takowanan** a maan? | （字面：你爱我什么？） | — |

## 语言对比 [COMPARISON]

| 特征 | 阿美语 | 汉语 | 英语 |
|------|--------|------|------|
| 指示词距离 | **三种**（近/远可见/远不可见） | 两种（这/那） | 两种（this/that） |
| 指示词格变 | **有**（主/属/斜，三格） | 无 | 无 |
| 地点指示词 | 与人物指示词不同形（ki-/ko-） | 加"里"（这里/那里） | 加"here"（here/there） |
| 所有格前置/后置 | 均可（mako a N / N no mako） | 仅前置（我的X） | 仅前置（my X） |
| 斜格人称 | **有**（-an结尾系列） | 用介词（对我/给你） | 用介词（to me） |
