## 一、速查卡片

### 字母关键点

```
汉语有对应: p t k m n ng s l / c h y w
汉语无对应: f d r x ' ^

重音 = 最后音节（加缀后移，是非问前移）
O + N + ko/ci/ca +主语 = "X是Y"
否定: Caay ko + N + ko/ci/ca +主语
```

### 动词 AF 标记速查

```
mi-    主动动作（捕鱼、盛饭、学习）
<om>   过程/姿态（哭、唱歌、跑）
ma-    状态/经历（饿、聪明、怕、跌倒）
∅     少数属性（高、痛）

V.AF + ko/ci + 主事者 + to + 受事者  ("X做Y")
```

## 三、答案

### 第一级
1. C（s 在汉语中有对应）
2. B（重音在最后音节 'is）
3. B（singsi 是谓语名词）
4. C（tomangic = t + om + angic）
5. D（∅ 是静态属性，非及物动作）
6. B
7. C（to 是斜格，标记受事者）
8. B（谓语在前）
9. A, C（c 和 s 在 i 前腭化）
10. B（否定名词谓语）

### 第二级
11.
| mi- | <om> | ma- | ∅ |
|-----|------|-----|---|
| mifoting, mililoc | romadiw, tomangic, comikay | matalaw, macahiw | takaraw |

- c) ma.fo.**ti'**（重音在 ti'）

### 第三级
16.
- 谓语类型：动词谓语（AF）
- 主语（主格）：ci Mayaw（主事者）
- 受事者（斜格）：to fonga
- 格位标记：ci标记主事者（主语），to标记受事者

17.
- a) 缺少 O → **O** singsi ci Mayaw.
- b) 格位标记反了 → Komaen **ci** Mayaw **to** fonga.（或保留 B 型词序：Komaen to fonga ci Mayaw.）
- c) macahiw 是不及物动词不需要 to → **Macahiw kako.**

18.
- a) Mifoting ci Panay.
- b) O tamdaw kako.
- c) Komaen kako to fonga.

- b) [ɕiʔnaw]（腭化 + 元音删除）
- c) [fafahejan]（滑音加插）
