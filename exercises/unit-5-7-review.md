## 一、速查卡片

### TAM 系统

```
模板: (Ca-重叠) + (na-) + 焦点标记 + 词根 + (-ay) + (=ho/=to)
       [非实现]    [过去]              [实现]   [未完成/完成]

na- = 过去     =to = 完成("了")     =ho = 未完成("还")
-ay = 实现(正发生/已发生)    Ca-重叠 = 非实现(未来/从未)
AF默认=非过去    PF ma-默认=过去    PF -en默认=非过去
```

### 否定与祈使焦点变换

```
肯定 → 否定/祈使:
  mi- → pi-     ma- → ka-     <om> → ka-<om>
  -en → -en(不变) / (ka-)...-en

否定词分工: caay/cai(叙述)  aka(祈使)  awa(存在/方位ira型)
```

### 句型速览

```
存在: Ira/Awa + ko + N + (i + 处所)
所有: Ira/Awa + ko + 所有物 + no/ni + 所有者  
方位: I + 方位名词 + ko/ci + 主语
祈使: pi-/ka-/-en + ...（肯定）; Aka + ...（否定）
使动: pa-（直接）; pa-pi-（弱使动"请/叫…做"）
关系子句: 仅主语可当中心语; AF/PF须加 -ay 或 Ca-重叠
```
