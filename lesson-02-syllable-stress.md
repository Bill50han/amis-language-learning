# 第2课：音节结构与重音——阿美语的"节奏感"

## 本课目标

学完本课后，你将能够：
1. 正确切分阿美语单词的音节
2. 判断任何单词的重音位置
3. 理解加词缀后重音为何会移动
4. 掌握最常用的三条音韵规则（腭化、元音删除、元音转滑音）

---

## 核心知识

### 1. 音节结构：最大 CVVC

阿美语音节以**辅音开始**为原则，基本结构如下：[FACT] pp.15-16

```
音节 = C (V) V (C)
       │  │  │  └── 韵尾辅音（至多一个）
       │  │  └───── 韵核元音（可双元音）
       │  └──────── 可选
       └─────────── 声母辅音（至多一个）
```

**最大音节**：CVVC（4个语音单位），其中 VV 为复元音，实际发音近似 VG 或 GV 组合。

[COMPARISON] 汉语音节最大为 CGVX（声母+介音+韵腹+韵尾，如"窗"chuang），阿美语音节**不允许辅音丛**（声母至多一个辅音），也不允许介音结构。所以阿美语的音节比汉语更"紧凑"。

### 2. 七种音节类型

[FACT] pp.15-16

| 类型 | 结构 | 出现条件 | 例词 | 音节切分 |
|------|------|---------|------|---------|
| **V** | 纯元音 | 有限分布 | ina "妈妈" | **i**.na |
| **VC** | 元音+辅音 | — | impic "铅笔" | **im**.pic |
| **CV** | 辅音+元音 | **最常见** | mata "眼睛" | **ma**.**ta** |
| **CVC** | 辅音+元音+辅音 | 常见 | tamdaw "人" | **tam**.**daw** |
| **CVV** (CGV) | 辅音+双元音(滑音+元音) | — | acya "赞叹声" | a.**cya** |
| **CVV** (CVG) | 辅音+双元音(元音+滑音) | — | pawli "香蕉" | **paw**.li |
| **CVVC** (CGVC) | 辅音+双元音+辅音 | — | micyan "煎" | mi.**cyan** |

[FACT] **音节一般以辅音开始**，元音不能单独构成音节（V型仅出现在特定位置）。pp.15-16

#### 音节切分练习

| 单词 | 音节切分 | 说明 |
|------|---------|------|
| fafahi | fa.fa.**hi** | 三个 CV 音节，重音在 hi |
| tamdaw | **tam**.**daw** | CVC + CVC，两个闭音节 |
| ngoso' | ngo.**so'** | CV + CVC（ng 是一个辅音字母） |
| mafoti' | ma.fo.**ti'** | CV + CV + CVC，重音在 ti' |
| romakat | ro.ma.**kat** | CV + CV + CVC，重音在 kat |

[FACT] ng 虽是两个字母，但表示一个音位 /ŋ/，在音节切分时算一个辅音。p.11

### 3. 重音规则：记住"最后一个音节"

#### 基本规则

**[FACT] 重音落在词的最后一个音节。** 这是阿美语最重要的韵律特征，也是区别于汉语（声调语言）的关键。pp.16-17

| 单词 | 音节 | 重读音节 | 汉译 |
|------|------|---------|------|
| ina | i.na | **na** | 妈妈 |
| mata | ma.ta | **ta** | 眼睛 |
| tamdaw | tam.daw | **daw** | 人 |
| fafahi | fa.fa.hi | **hi** | 太太 |
| romadiw | ro.ma.diw | **diw** | 唱歌 |

#### 规则2：加词缀，重音跟着走

[FACT] 当单词加上词缀（前缀或后缀），重音随之移动，仍落在新词的最后一个音节。p.16

| 词根 | 加缀后 | 变化说明 |
|------|--------|---------|
| **kaen** "吃" | kaen**en** "被吃" | 加后缀 -en，重音移至 -en |
| **foti'** "睡" | ma.**foti'** "睡觉" | 加前缀 ma-，重音仍在最后 |
| **nanam** "学习" | na.na.**man** "被学习的地方" | 加 -an，重音移至 -an |

[COMPARISON] 这是汉语母语者需要特别注意的。汉语词没有"重音随词缀移动"的概念。英语中也有类似现象（如 photo → photógraphy），但英语更复杂且不规律。阿美语这条规则是**完全规律的**。

#### 规则3：是非问句——重音前移

[FACT] 在是非问句中，最后一个词的**倒数第二音节**语调升高（↗），最后音节降至平调，重音移至倒数第二音节。p.17

| 句式 | 例句 | 重音 |
|------|------|------|
| 直述句 | Mafana' kiso a romadiw. "你会唱歌。" | ro.ma.**diw** ↘（降调） |
| 是非问 | Mafana' kiso a romadiw? "你会唱歌吗？" | ro.**ma**.diw ↗→ |

[COMPARISON] 汉语疑问句用"吗"或句尾升调，阿美语通过**重音位移+语调变化**。这与英语的疑问句升调类似但机制不同——阿美语连重音位置都变了。

#### 规则4：强调语气——倒数第二音节延长

[FACT] 为表强调，倒数第二音节元音延长，重音也移至该音节。pp.16-17

### 4. 最重要的三条音韵规则

[FACT] 教材列出七条音韵规则（pp.16-17），对初学者而言，以下三条最重要的是实际中最常遇到的：

#### 规则1：腭化（Palatalization）

```
/ts, s/ → [tɕ, ɕ] / __ /i/
c, s 在 i 前 → 变成舌面音
```

| 拼写 | 实际读音 | 说明 |
|------|---------|------|
| **ci**ma | [**tɕi**ma] | c→[tɕ]，类似中文"几" |
| po**si** | [pʊ**ɕi**] | s→[ɕ]，类似中文"西" |
| ca**ci**yaw | [tsa**tɕi**jaw] | 第一个 ci 腭化 |
| **si**wa | [**ɕi**wa] | s→[ɕ] |

[FACT] 这就是为什么 cima "谁" 听起来像"吉马"而不是"次马"。**si 音串不出现**——[si] 总是变成 [ɕi]。pp.16-17

[COMPARISON] 类似汉语拼音 j/q/x 只能接 i/ü 不能接 a/o/e 的分布限制，但方向相反——阿美语是 c/s 在 i 前自动变，汉语是 j/q/x 本来就是舌面音。

#### 规则6：元音删除（/ə/-Deletion）

```
/ə/ → ∅ / C__ （非重音节）
e 在非重音节中，前面有辅音时 → 删除
```

| 拼写 | 实际读音 | 说明 |
|------|---------|------|
| **te**fo' | [**t**foʡħ] | 重音在 fo'，te 非重音→e 删除 |
| si**^e**naw | [ɕiʔnaw] | 重音在 naw，^en 非重音→e 删除 |

[FACT] 这是为什么有些单词听起来比拼写"少了一个音节"。e (/ə/) 在非重音位置经常被吞掉。p.17

[COMPARISON] 类似英语 "family" 读成 [fæmli]（省略中间的 ə）或汉语"豆腐"dòufu→dòuf。阿美语的 e-删除规则与此类似但更系统。

#### 规则5：滑音加插（Glide-Insertion）

```
/i, o/ → [ij, ow] / __ V
i 或 o 后面紧接另一元音时 → 中间插入滑音 [j] 或 [w]
```

| 拼写 | 实际读音 | 说明 |
|------|---------|------|
| fafa**hi**an | [fafa**hej**an] | hi 后接 an，i→[j] 滑音插入 |

[FACT] 这条规则解释了为什么 fafahian 读作"[fafahejan]"而不是"[fafahian]"——两个元音之间自动插入了一个滑音。p.17

[FACT] 教材七条规则中的规则3（元音转滑音：/i,o/→[j,w]/V__）也会产生滑音，如 saomah→[sawmah]，但本课仅选取最常见三条讲解。完整七条参见教材 pp.16-17 或知识库 ch02。

## 综合运用：从拼写到实际发音

以一个稍长的词为例，展示所有规则的叠加效应：

```
maisi'（秀姑峦阿美语："哭过之后眼睛有泪水"）
拼写：ma.i.si'
音节：ma.i.si'
重音：si'
腭化：si' → [ɕiʡħ]（规则1腭化 + 规则2送气）
元音转滑音：mai → [maj]（规则3：i 前有 a，i→[j]）
实际读音：[majɕiʡħ]

> **注意**：上例用到了教材七条规则中的规则2（辅音送气）和规则3（元音转滑音），这两条不在本课重点讲解的三条规则中。完整七条规则见教材 pp.16-17 或知识库 ch02-phonology.md。
```

## 语言对比 [COMPARISON]

| 特征 | 阿美语 | 汉语 | 英语 |
|------|--------|------|------|
| 韵律系统 | **重音语言** | **声调语言** | 重音语言 |
| 重音位置 | 最后音节（规律） | 无重音概念 | 不规则 |
| 加缀后 | 重音随之后移 | 无 | 常改变重音（不规则） |
| 音节复杂度 | 简单（最大CVVC） | 中等（最大CGVX） | 复杂（CCCVCC） |
| 辅音丛 | 不允许 | 不允许 | 允许 |

[FACT] 阿美语非声调语言，其韵律完全由重音和语调控制。汉语母语者需要"关掉"声调直觉，转而关注重音位置。pp.16-17
