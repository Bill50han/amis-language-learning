# 第18课：使动结构——"让/叫/使"某人做某事

## 本课目标

学完本课后，你将能够：
1. 理解阿美语使动结构三种类型（词汇式/构词式/句法式）
2. 使用 pa- 前缀表达"使/让/叫"（最常见的使动方式）
3. 区分 pa-（直接使动）和 pa-pi-（弱使动"请/叫…做"）
4. 理解三论元使动句的主语选择

---

## 核心知识

### 1. 什么是使动结构？

[FACT] 使动结构 = 某个动作导致了另一个动作/事件/状态的发生。比非使动结构多一个论元——**使动者**。pp.83

```
非使动:   komaen ci Mayaw.  "Mayaw在吃。"
使动:     pa-kaen-en ni ina ci Mayaw.  "妈妈在喂Mayaw。"
           ↑       ↑       ↑
          使-吃    使动者   受动者（"被喂的人"）
```

[COMPARISON] 汉语用"让/叫/使/把"（"我让/叫他吃饭"），英语用 make/let/have。阿美语主要通过 **pa- 前缀**系统化表达。

### 2. 三种使动类型

[FACT] pp.83-88

| 类型 | 机制 | 特征 | 例 |
|------|------|------|-----|
| **词汇式** | 动词本身带使动义 | 数量少，不需 pa- | feli "给" (=使有), pacok "宰杀" (=使死) |
| **构词式** | pa- 前缀 | **最常见** | pa-kaen "喂" (=使吃) |
| **句法式** | pa-动词 + 其他动词 | 复杂句 | patangic "强迫" + V |

### 3. pa- 前缀：最核心的使动标记

[FACT] pa- 是最常见、最能产的使动标记。加在不同词类上有不同效果：pp.83-87

| 词根类别 | pa- 语义 | 例 |
|---------|---------|-----|
| **动词** | "使做X" | kaen "吃" → **pa**-kaen "喂"（使吃） |
| **名词** | "使有X / 加X" | dateng "菜" → **pa**-dateng "加菜" |
| **形容词** | "使变成X" | kaeped "软" → **pa**-kaeped "使变软" |
| **交易动词** | 改变方向 | cakay "买" → **pa**-cakay "卖"；caliw "借入" → **pa**-caliw "借出" |

[FACT] 交易动词加 pa- 后改变动作方向：买→卖、借入→借出。这是 pa- 的一个特殊子系统。p.84

#### 例句

| 阿美语 | 结构 | 汉语 | 来源 |
|--------|------|------|------|
| Pa-kaen-en ni ina ko wawa. | PF: pa-吃-en + ni+妈妈 + ko+孩子 | 妈妈在喂孩子。 | p.87 |
| Pa-rakat-en nira koni a paliding. | PF: pa-走-en + 他 + ko+车 | 他开这辆车。(=使车走) | p.87 |
| Pa-dateng-en ko hemay. | PF: pa-菜-en + ko+饭 | 给饭加菜。 | p.87 |

### 4. pa-（直接使动）vs pa-pi-（弱使动）

[FACT] **关键区分**：pp.87-88

| 标记 | 使动类型 | 语义 | 受动者 |
|------|---------|------|--------|
| **pa-** | 直接使动 | 直接使对象发生变化 | 物/人（直接影响） |
| **pa-pi-** | 弱使动 | "请/叫/让…做某事" | 人（间接，受动者自己去做） |

| 阿美语 | 汉语 |
|--------|------|
| **Pa**-rakat-en nira koni a paliding. | 他**开**这辆车。（直接使车走） |
| **Pa-pi**-rakat-en nira ci Panay. | 他**请**Panay去走路。（*使Panay走路） |

[FACT] pa-pi- 中的 pi- 是对应 mi-动词的形式——这一规律现在应该很熟悉了。p.87-88

### 5. 句法式使动——带两个动词的复杂句

[FACT] 句法式使动用 pa-动词 + 受动者 + 另一动词：pp.88

| 阿美语 | 结构 | 汉语 |
|--------|------|------|
| Patangic kako ci Akian tayra i Taypak. | pa-哭 + 我 + ci+Aki + 去+在+台北 | 我强迫Aki去台北。 |

[FACT] 受动者（Aki）同时是第二个动词（tayra "去"）的主事者——形成**枢轴结构**（pivot）。p.88

### 6. 三论元使动句的主语选择

[FACT] 当使动句有三个参与者（使动者、受动者、受役物），PF句的主语选择有三种可能：pp.89-91

| 类型 | PF主语可选谁 | 例词 |
|------|-----------|------|
| **仅受动者可作主语** | 受动者✓ 受役物✗ | pa-kaen "喂"（主食=被喂的人） |
| **仅受役物可作主语** | 受动者✗ 受役物✓ | pa-resap "使散播"（主食=被传播的话） |
| **两者皆可作主语** | 受动者✓ 受役物✓ | pa-kalat "给…咬" |

[INFERENCE] 主语选择可能与动词的语义角色层级（有生性/受影响度）有关，教材未给出统一规则。pp.89-91

## 句型公式卡片

```
┌──────────────────────────────────────────────┐
│  pa- 使动结构                                  │
│                                              │
│  pa- + 动词:   "使做X"（直接）                  │
│  pa- + 名词:   "使有X / 加X"                   │
│  pa- + 形容:   "使变X"                         │
│  pa- + 交易:   "方向反转"（买→卖，借入→借出）     │
│                                              │
│  pa-pi- :   弱使动 "请/叫…做"                   │
│                                              │
│  双论元: pa-V-en + ni/no + 使动者 + ko + 受动者  │
│  三论元: pa-V-en + ni/no + 使动者 + ko + 主食    │
│          + to/ci…-an + 另一宾语                 │
└──────────────────────────────────────────────┘
```
