# 第7课：人称代名词（上）——"我/你/他"的主格与属格

## 本课目标

学完本课后，你将能够：
1. 使用完整的主格人称代名词（kako / kiso / cingra / kita / kami / kamo / cangra）
2. 理解并正确运用包含式（kita）和排除式（kami）——汉语没有的区分
3. 使用附著式属格代名词（=ako / =iso / =ita 等）表达"我的/你的/他的"
4. 区分自由式和附著式的使用场景

---

## 核心知识

### 1. 人称代名词系统概览

[FACT] 阿美语人称代名词有三套形式：**主格自由式**（做主语）、**属格附著式**（"的"，依附在名词后）、**属格/所有格自由式**（可独立使用）。本课先学前两套。pp.47-52

[FACT] 所有人称代名词**无性别之分**——cingra 既可以是"他"也可以是"她"。pp.48

### 2. 主格代名词——"我/你/他"做主语

[FACT] 主格自由式代名词直接做主语，不需要再加 ko 或 ci：pp.48-49

| 数/人称 | 阿美语 | 汉语 |
|---------|--------|------|
| 单一 | **kako** | 我 |
| 单二 | **kiso** | 你 |
| 单三 | **cingra** / cira | 他 / 她 |
| 复一**含** | **kita** | 咱们（包含听者） |
| 复一**排** | **kami** | 我们（不包含听者） |
| 复二 | **kamo** | 你们 |
| 复三 | **cangra** / caira | 他们 / 她们 |

使用示例：

| 阿美语 | 汉语 |
|--------|------|
| Komaen **kako** to fonga. | 我吃地瓜。 |
| Mafoti' **kiso**. | 你在睡觉。 |
| O singsi **cingra**. | 他/她是老师。 |
| Mifoting **kita**. | 咱们在捕鱼。 |
| Mataneng **kami**. | 我们很聪明。（不含你） |
| C\<om\>ikay **kamo**. | 你们在跑步。 |

[FACT] 主格代名词直接替代 ko/ci + 名词的位置，不需要额外加格位标记。pp.48-49

### 3. ★ 核心概念：包含式 vs 排除式

[FACT] 第一人称复数分为两种形式，这是阿美语**最重要的代名词语法特征**：pp.48

| 形式 | 阿美语 | 含义 | 用法 |
|------|--------|------|------|
| **包含式** | **kita** | "咱们" = 我 + 你（们） | 听者包含在内 |
| **排除式** | **kami** | "我们" = 我 + 其他人，但不包括你 | 听者不包含在内 |

**举例说明区别**：

| 场景 | 用 kita（含）还是 kami（排）？ |
|------|---------------------------|
| 你对朋友说："我们去吃饭吧。" | **kita**——你邀请对方一起，包含听者 |
| 你对外人说："我们不喝酒。" | **kami**——你在陈述你和你的群体的习惯，不包含外人在内 |
| 你对外人说："我们要走了。" | **kami**——你和你的同伴要走，不包括听者 |
| 你对同伴说："咱们走吧。" | **kita**——你和同伴一起行动 |

[COMPARISON] 汉语普通话没有"包含/排除"语法区分。"我们"可以包含听者也可以不包含，靠语境判断。部分汉语方言有类似区分（如北京话"咱们"=包含 vs "我们"=排除），可以借助这个来理解——**kita ≈ 咱们，kami ≈ 我们**。

英语同样没有此区分（都是 we/us）。

### 4. 附著式属格代名词——"我的/你的"（简式）

[FACT] 附著式属格代名词**必须依附**在名词后面，不能独立使用。这是表达领属最常用的方式。pp.49-51

| 数/人称 | 阿美语 | 汉语 |
|---------|--------|------|
| 单一 | **=ako** | 我的 |
| 单二 | **=iso** | 你的 |
| 单三 | nira | 他的/她的 |
| 复一含 | **=ita** | 咱们的 |
| 复一排 | niyam | 我们的（排） |
| 复二 | namo | 你们的 |
| 复三 | nangra / naira | 他们的 |

**用法**：直接附着在名词后面

```
wawa + =ako  →  wawa=ako   "我的孩子"
mama + =iso  →  mama=iso   "你的爸爸"
loma' + nira →  loma' nira "他的房子"
```

注意：第三人称 nira 前面通常不写等号（=），写为独立词，但语法功能同附着式。

| 阿美语 | 结构 | 汉语 |
|--------|------|------|
| wawa **=ako** | 孩子 + 我的 | 我的孩子 |
| mama **=iso** | 爸爸 + 你的 | 你的爸爸 |
| loma' **nira** | 房子 + 他的 | 他的房子 |
| foting **=ita** | 鱼 + 咱们的 | 咱们的鱼 |
| singsi **niyam** | 老师 + 我们的(排) | 我们的老师 |
| payso **namo** | 钱 + 你们的 | 你们(们)的钱 |

[FACT] 附著式属格 =ako /=iso /=ita 的等号表示它们是**依附词**（clitic），不能独立存在，必须紧跟前面的名词。pp.49-51

### 5. 自由式属格 vs 附著式属格

[FACT] 部分人称还有**自由式属格**（可独立使用），与附著式的功能基本重叠：p.49-51

| 人称 | 附著式 | 自由式 |
|------|--------|--------|
| 单一 | =ako | nira |
| 单二 | =iso | nira |
| 单三 | nira | nira |
| 复一排 | niyam | niyam |
| 复二 | namo | namo |

[FACT] 大部分人称的属格只有一种形式（附著式和自由式同形或只有附著式）。第一、第二人称单数的附著式（=ako, =iso）最为常用。p.49-50

[FACT] 注意：第一/第二人称单数没有独立的自由式属格，借用第三人称的 nira。需依语境区分指称对象。p.49-51

## 语言对比 [COMPARISON]

| 特征 | 阿美语 | 汉语 | 英语 |
|------|--------|------|------|
| 包含/排除 | **有**（kita/kami） | 无（"咱们/我们"近义但不系统） | 无（we） |
| 性别区分 | **无**（cingra=他/她） | 有（他/她/它） | 有（he/she/it） |
| 附着式属格 | **有**（=ako, =iso） | 有（"我的"可缩略为"我"） | 有（my, your） |
| 格位变化 | **有**（主格/属格/斜格不同形） | 无（"我"不变） | 部分有（I/me/my） |
| 自由 vs 附着 | 严格区分 | 无形式区分 | 无形式区分 |

[FACT] 阿美语代名词系统比汉语复杂的三个维度：① 包含/排除 ② 格位变体 ③ 自由/附著。但它没有性别和敬语（比日语简单）。pp.48-52
