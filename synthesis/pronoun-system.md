# 阿美语代名词系统总览

> 来源：第5章 | 综合日期：2026-05-19

## 1. 人称代名词完整表

### 自由式

| 数/人称 | 主格 | 属格 | 所有格 | 斜格 |
|---------|------|------|--------|------|
| 单一 | **kako** | nira | **mako** | takowanan/takonan |
| 单二 | **kiso** | nira | **miso** | tisowanan/kisonan |
| 单三 | **cingra/cira** | **nira** | mira | cingraan/ciraan |
| 复一含 | **kita** | — | **mita** | kitanan/titaanan |

| 复二 | **kamo** | **namo** | **namo** | tamoanan |
| 复三 | **cangra/caira** | **nangra/naira** | mira | cangraan/cairaan |

|---------|------|
| 单一 | **=ako** |
| 单二 | **=iso** |
| 单三 | =nira |
| 复一含 | **=ita** |
| 复一排 | =niyam |
| 复二 | =namo |
| 复三 | =nangra/=naira |

## 2. 关键特征

| 特征 | 说明 |
|------|------|
| 无性别区分 | 所有代名词无性别之分 |
| 包含/排除 | [FACT] 第一人称复数分**包含式**(kita, =ita, mita)和**排除式**(kami, niyam) |
| 附著式限制 | [FACT] 附著式只出现在属格(=ako, =iso, =ita, niyam, namo, nira/naira) |
| 第三人称 | [FACT] 形式与指示代名词有关 |
| 斜格特征 | [FACT] 均以 -an 结尾；多形式为方言差异 |

## 3. 属格 vs 所有格 对比

| | 属格（附著式 =ako） | 属格（自由式 nira） | 所有格（mako） |
|------|------|------|------|
| 独立性 | 必须依附寄主词后 | 可独立 | 可独立 |
| 位置 | N=ako | (同附著式) | **mako a** N / N **no mako** |
| 与格标记共现 | 否 | 否 | 必须（+ ko/no） |
| 可用的格功能 | 仅属格 | 仅属格 | 通过格位标记决定(ko mako=主格, no mako=属格) |

## 4. 指示代名词

### 标示人/事/物

| 距离 | 主格 | 属格 | 斜格 | 词尾特征 |
|------|------|------|------|---------|
| 近 | **koni** | noni | toni | /ni/ |
| 远+可见 | **kora**/koraw | nora/noraw | tora/toraw | /ra(w)/ |
| 远+不可见 | **koya** | noya | toya | /ya/ |

### 标示地点

| 距离 | 主格 | 属格 | 斜格 |
|------|------|------|------|
| 近 | **kini** | ninian | **tini** |
| 远+可见 | **kira**/kiraw | nira/niraw | **tira**/tiraw |
| 远+不可见 | **kiya** | niya | **tiya** |

### 用法规则

| 规则 | 说明 |
|------|------|
| 独立使用 | 可代替整个名词组 |
| 修饰名词 | 指示词 + (a) + 名词。koni + a → **kona** |
| 变体 | 近指有重疊变体(konini)和后缀变体(konian)；远指仅-an变体(koraan/koyaan) |
| 地点斜格 | 只有 + i 时才标示地点：i tini "在这里" |
| tiya特殊用法 | i tiya ho = "从前" |
