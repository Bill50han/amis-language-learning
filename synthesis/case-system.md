# 阿美语格位系统总览

> 来源：第5章为主，第4、6章为辅 | 综合日期：2026-05-19

## 1. 格位系统总表

阿美语为**三分格位**格局（主格/属格/斜格），按名词类型分两套，人称专有再分单复数：

| 名词类型 | 主格(NOM) | 属格(GEN) | 斜格(OBL) |
|---------|----------|----------|----------|
| **普通名词** | **ko** | **no** | **to** |
| **人称专有·单数** | **ci** | **ni** | **ci ...-an** |
| **人称专有·复数** | **ca** | **na** | **ca ...-an** |

[FACT] 深层分析：ko = k(主格) + o(普通名词标记)；no = n(属格) + o；to = t(斜格) + o。（Huang 1995）

## 2. 各格位功能对照

### 主格 (ko, ci, ca)

| 功能 | 例句模式 |
|------|---------|
| 标示AF句主事者 | V.AF + **ko/ci** + N(主事者) + to + N(受事者) |
| 标示PF句受事者 | V.PF + no/ni + N(主事者) + **ko/ci** + N(受事者) |
| 标示工具施用主语(工具) | V.sa- + no + N + **ko** + N(工具) |
| 标示处所施用主语(地点) | V.-an + no + N + **ko** + N(地点) |
| 标示名词谓语句主语 | O + N + **ko/ci** + N(主语) |
| 标示存在句主语 | Ira **ko** + N... |

[FACT] 主格标记与动词焦点词缀构成**呼应关系**。**ko**可标示定指或非定指。

### 属格 (no, ni, na)

| 功能 | 例句模式 |
|------|---------|
| 标示领属/所有者 | N + **no/ni** + 所有者 |
| 标示非AF句主事者 | V.PF + **no/ni** + N(主事者) + ko + N(受事者) |

### 斜格 (to, ci/ca...-an)

| 功能 | 标记 | 限制 |
|------|------|------|
| AF句受事者 | to | — |
| 三论元接受者/受动者 | to 或 ci...-an | — |
| 时间/原因 | to | — |
| 地点 | **只能用 -an**（+ i），**不能用to** | to≠方位格 |
| 工具（非主语时） | to | — |

## 3. 格位-焦点-词序 互动关系

```
主事焦点句(AF):
  [V.AF] - [主事者·主格(ko/ci)] - [受事者·斜格(to)]
  Komaen    ci Mayaw              to fonga.
  "Mayaw吃地瓜。"

受事焦点句(PF):
  [V.PF] - [主事者·属格(no/ni)] - [受事者·主格(ko/ci)]
  Makaen    ni Mayaw              ko fonga.
  "地瓜被Mayaw吃了。"

工具施用句:
  [V.sa-] - [主事者·属格] - [工具·主格(ko)]
  Sapicekiw ni ina                ko rinom...
  "妈妈用针挑挖..."

处所施用句:
  [V.-an] - [主事者·属格] - [处所·主格(ko)]
  Pitangtangan ni ina             ko sefi ni Panay.
  "妈妈在Panay的厨房煮..."
```

## 4. 名词类别标记

| 形式 | 功能 | 例句 |
|------|------|------|
| **o** | 普通名词谓语标记 | **O** singsi ci Mayaw. "Mayaw是老师。" |
| **ci** | 人称专有单数谓语标记 | — |
| **ca** | 人称专有复数谓语标记 | **Ca** Panay kora pikadafoan... "入赘的是Panay她们家。" |

[FACT] 本书采用"名词类别标记"分析（非"中性格"），认为标示谓语非格位功能。

## 5. 关键语法约束

1. [FACT] 受事焦点句中，主事者（属格）**只能出现在受事者（主格）之前**
2. [FACT] 斜格 **to** 不能标示地点——地点用介系词 **i**
3. [FACT] 处所斜格 **-an** 可单独表地点，也常与 i 共现：(i) ci ...-an
4. [FACT] 格位标记出现在名词组最前端，不可省略
