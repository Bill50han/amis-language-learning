# 阿美语构词规则总览

> 来源：第3章 | 综合日期：2026-05-19

## 1. 三大构词方法

```
构词方法
├── 加缀法 (Affixation) — 最丰富
│   ├── 前綴：ci-, hali-, pa-, sa-, tala-, pi-, ka-, paka-, mi-, ma-
│   ├── 中綴：<om> (仅一个)
│   ├── 後綴：-ay, -an, -en
│   └── 環綴：pi-...-an, sa-...-an
├── 重疊法 (Reduplication)
│   ├── 詞彙化重疊：rohroh, dokdok (不可分解)
│   ├── CVCV雙音節重疊：wawa-wawa, raka-rakat
│   ├── Ca重疊：ta-tosa, ma-mi-dotoc, ma-ca-coli
│   └── 特殊形式重疊：tamdamdaw (极少用)
└── 複合法 (Compounding)
    ├── a-连接：fafahian a kaka "姐姐"
    └── 属格连接：'alo no fo'is "银河"
```

## 2. 加缀法：派生方向

```
名词 → 动词：
  ci- + N：ci-payso "有钱"
  sa- + N：sa-^epah "酿酒"
  mi- + N：mi-foting "捕鱼"

动词 → 名词：
  pi- + V：pi-cocok "打针(行为)"
  -ay + V：mifoting-ay "渔夫"
  pi-...-an + V：pi-codad-an "学校"
  ka- + V：ka-etip "西边"

静态动词 → 使动动词：
  mi- + 静V：mi-lasang "使醉"
  -en + 静V：fa'edet-en "加热"
  pa- + 静V：pa-kaeped "使变软"
```

## 3. 重疊法：语义功能对照

| 重疊类型 | 形式 | 名词 | 动态动词 | 静态动词 |
|---------|------|------|---------|---------|
| CVCV | 双音节重叠 | 复数/数量多 | 持续/反复 | 程度加强/分布 |
| Ca | C₁+a | 表人个数(+数词) | 非实现/未来 | 表人个数 |
| ma-Ca- | ma-+Ca | — | 交互动作 | — |
| 特殊 | C₂.C₃V₂插入 | tamdamdaw "人人" | — | — |

### Ca重叠的多功能性（跨章综合）

```
Ca重疊 =
├── Ch3: 与数词连用 → 表人个数 (ta-tosa "两个人")
├── Ch6: 与动词连用 → 非实现/未来 (ma-mi-dotoc "将要继承")
├── Ch6: Ca重疊 + 动词 → 构成TAM非实现标记
├── Ch14: 关系子句中 → 非实现貌
└── Ch16: ma-Ca- + 词根 → 交互结构 (ma-ca-coli "互相吵架")
```

[INFERENCE] Ca重叠的核心语义可能是「非实际/非现实」——表人(尚未确定为特定个体)、未来(尚未发生)、交互(双向非单向)。强度：中等，教材未明确统一说明。

## 4. 复合词

| 连接方式 | 结构 | 例 | 语意 |
|---------|------|-----|------|
| a（连繫词） | N + a + N | fafahian a kaka | "姐姐" (女人+兄姐) |
| a | N + a + N | kilang a kayakay | "木桥" (树+桥) |
| no/ni（属格） | N + no/ni + N | 'alo no fo'is | "银河" (河流+星星) |
| ni | N + ni + 人名 | teker ni Idek | "彩虹" (陷阱+Idek) |

## 5. 借词来源分布

| 来源 | 数量 | 典型领域 | 例词 |
|------|------|---------|------|
| 日语 | 最多 | 交通工具、电器、制度、饮食 | takosi"计程车", dingki"电", singsi"老师", ucya"茶" |
| 闽南语 | 次多 | 饮食、器物 | ising"医生", piyang"饼", ekim"金子" |
| 汉语 | 较少 | 学校、制度 | congtong"总统", kawcong"高中" |
| 英语 | 极少 | 宗教 | kolismas"圣诞节(Christmas)" |

## 6. 词类系统

| | 开放类（实词） | 封闭类（虚词） |
|------|------|------|
| 成员 | 名词、动词 | 格位标记、代名词、指示词、疑问词、连接词、助词、惊叹词 |
| 特征 | 可有新词产生 | 数量固定，不易产生新词 |

[FACT] 修饰概念（大/小/新/旧/快/慢）在阿美语中用**静态动词**表达，非形容词。
