# 阿美语句型结构总览

> 来源：第4章（基本句型）、第7-16章（各句类） | 综合日期：2026-05-19

## 1. 基本语序原则

[FACT] 阿美语为**谓语在前(predicate-initial)** 语言。

```
句子 = 谓语 + 非谓语名词组(主语 + 其他成分)
```

## 2. 谓语类型

| 类型 | 标记 | 结构 | 例句模式 |
|------|------|------|---------|
| **动词谓语** | 焦点词缀(AF/PF) | V + NP... | Komaen ci Mayaw to fonga. |
| **名词谓语** | o/ci/ca | o/ci/ca + N + 主语 | O singsi ci Mayaw. |
| **介系词谓语** | i | i + 处所 + 主语 | I kawili no loma' ko cawka. |
| **存在/所有/方位** | ira/awa | ira/awa + ... | Ira ko ceka i tarodo' ako. |

## 3. 四大基本句型词序

### 直述句

| 子型 | 词序 |
|------|------|
| AF-单论元 | V.AF - 主语(主格) |
| AF-双论元 | V.AF - 主事者(主格) - 受事者(斜格) |
| PF-双论元 | V.PF - 主事者(属格) - 受事者(主格) |
| 三论元(使动) | V - 使动者 - 受动者 - 受役物 |

### 否定句

| 谓语句型 | 词序 |
|---------|------|
| 动词谓语否定 | **caay/cai** + V(非限定形) + ... |
| 名词谓语否定 | **caay/cai** + ka-o/ko + N + 主语 |
| 祈使否定 | **aka** + V(祈使形) + ... |
| 存在否定 | **awa** + ... |

### 疑问句

| 类型 | 形成方式 |
|------|---------|
| 是非问 | 语调改变(↗倒数第二音节) 或 句尾+haw/saw |
| 选择问 | 选项1 + **ano ca i** + 选项2 |
| 讯息问 | 疑问词(cima/maan等)通常句首，可留句中 |

### 祈使句

| 焦点 | 肯定 | 否定 |
|------|------|------|
| AF | **pi-**/ka- + 词根! | **aka** pi-/ka- + 词根! |
| PF | 词根-**en**! | **aka** ka-词根-en! |
| 工具施用 | sa-pi/ka-词根-**en**! | — |
| 叮咛型 | 词根 + **han**! | 词根 + **aka han**! |

## 6. 句型-焦点-格位 整合规则

```
所有句子类型必须满足：
  焦点词缀 ↔ 主格标记(ko/ci/ca) ↔ 主语语义角色

AF句 → 主语 = 主事者 → 主格标记 主事者
PF句 → 主语 = 受事者 → 主格标记 受事者
工具施用 → 主语 = 工具/原因 → 主格标记 工具/原因
处所施用 → 主语 = 处所/受事/目标 → 主格标记 处所/受事/目标
```

## 7. 否定-祈使动词形式的统一分析

```
肯定叙述 ⊣ 否定叙述 ⊣ 肯定祈使
   mi-    →    pi-    →    pi-
   ma-    →    ka-    →    ka-
  <om>    →  ka-<om>  →  ka-<om>
   -en    →  (ka)-en  →    -en
   sa-    →   ka-sa-  →  sa-...-en
   -an    →  ka-...-an →   —
```

[INFERENCE] pi-/ka- 是阿美语的**非限定(non-finite)动词标记**，同时用于否定句和祈使句。依据：否定句中caay/cai要求非限定形，祈使句要求非限定形——两者使用相同的pi-/ka-标记。
