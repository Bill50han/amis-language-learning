# 阿美语焦点系统总览

> 来源：第6章为主，第8章（祈使焦点）、第14章（关系子句焦点）为辅 | 综合日期：2026-05-19

## 1. 核心分析框架

本书采用**两焦分析**（vs 传统四焦分析）：

```
焦点系统
├── 主事焦点 (AF): mi-, ma-, <om>, ∅
│   └── 语法不及物结构。主语=主事者（动作者/经验者/位置客體）
│
└── 受事焦点 (PF): 语法及物结构。主语=受事者（或其扩展）
    ├── 非施用结构: ma-, -en
    ├── 工具施用结构: sa- (+pi-/ka-/pa-)
    └── 处所施用结构: -an (+mi-/ni-/ka-/pi-/<om>)
```

[FACT] 传统工具/处所焦点 → 本书重分析为受事焦点的**施用结构**。关键证据：
- 工具/处所标记可与PF标记共现（ma-sa-, sa-...-en）
- 在关系子句中无时貌区分（vs AF/PF必须加-ay或Ca重叠）
- 无明显的时貌讯息

## 2. AF标记对比

| 标记 | 动态性 | 典型词类 | 例 |
|------|--------|---------|-----|
| **mi-** | 最高（动作） | 动态动词 | mi-foting "捕鱼", mi-'atal "盛" |
| **<om>** | 中等（过程） | 过程动词/姿态 | t\<om\>angic "哭", r\<om\>adiw "唱歌" |
| **ma-** | 低（状态/被动） | 静态动词 | ma-taneng "聪明", ma-tolo' "跌倒" |
| **∅** | 静态（属性） | 少数静动 | takaraw "高" |

## 3. PF标记对比

| 标记 | 事件性质 | 主事者限制 | 可用句型 | 特殊用法 |
|------|---------|----------|---------|---------|
| **ma-** | 已发生，受事受影响 | 可无生命 | 仅陈述句 | — |
| **-en** | 有定受事，终点性，意志力 | 必须属人 | 陈述+祈使 | 与静动连用=使动 |

## 4. 施用结构

### 工具施用 (sa-)

| 成分 | 形式 | 功能 |
|------|------|------|
| sa- + pi- | mi-动词对应 | 突显工具/原因为主语 |
| sa- + ka- | ma-/静态动词对应 | 同上 |
| sa- + pa- | 使动动词对应 | 同上 |

[FACT] 工具施用较常以**等同句**呈现（O + 工具 + ko + sa-pi/ka/pa-...）。

### 处所施用 (-an) — 三种子类

| 子类 | 动词标记 | 突显主语 |
|------|---------|---------|
| 处所-目标 | mi-...-an / ni-...-an | 目标 |
| 处所-受事 | ni-...-an, <om>...-an, ka-...-an | 受事者 |
| 处所-地点 | pi-...-an, ka-...-an, ka-<om>-...-an | 地点/时间 |

## 5. 焦点-TAM互动

| | 过去 | 进行(非过去) | 未来(非过去) |
|------|------|------|------|
| AF | — | mi-, <om>, ma- | mi- |
| PF | ma- | — | -en |

无TAM标记时，焦点标记本身暗示时貌。

## 6. 非直述句中的焦点变换

### 祈使句焦点 (Ch8)

| 叙述句 | 祈使句 | 变换规则 |
|--------|--------|---------|
| mi- | **pi-** | mi→pi (如 mi-foting → pi-foting!) |
| ma-, <om> | **ka-** | ma→ka (如 ma-tayal → ka-tayal!) |
| -en | **-en** (不变) | (如 kaen-en!) |
| sa- | **sa-...-en** | (如 sa-pi-litek-en!) |

### 否定句焦点 (Ch10)

| 叙述句 | 否定句(caay/cai后) |
|--------|-------------------|
| mi- | **pi-** |
| ma-, <om> | **ka-** |
| ma- (PF) | **ka-** |
| -en (PF) | **(ka-)**...**-en** |
| sa- | **ka-sa-** |
| -an | **ka-...-an** |

### 关系子句焦点 (Ch14)

| 焦点 | 实现貌 | 非实现貌 |
|------|--------|---------|
| AF | mi-/ma-...**-ay** | **Ca**重疊+mi-/ma- |
| PF | ma-...**-ay** | **Ca**重疊+ma-/-en |
| 工具/处所 | 不变（无时貌区分） | 不变 |

## 7. 焦点-格位呼应总结

```
AF句:  V.AF  + ko/ci + 主事者 + to + 受事者
PF句:  V.PF  + no/ni + 主事者 + ko/ci + 受事者
工具:  V.sa- + no/ni + 主事者 + ko + 工具     (+ to + 受事者)
处所:  V.-an + no/ni + 主事者 + ko + 处所/受事 (+ to + ...)
```
