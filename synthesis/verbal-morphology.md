# 阿美语动词形态系统总览

> 来源：第3章（词缀）、第6章（焦点+TAM）、第9章（使动）、第8章（祈使） | 综合日期：2026-05-19

## 1. 动词模板

```
(Ca重疊) - (na-) - 焦点/使动前缀 - <中缀> - 词根 - (-an/-en/-ay) - (=to/=ho)
 [非实现]  [过去]   [pi-/mi-/ma-/pa-/sa-/ka-] [<om>]         [施用/焦点/实现]  [TAM依附词]
```

## 2. 词缀清单（按功能）

### 动词化词缀

| 词缀 | 类型 | 功能 | 来源章 |
|------|------|------|--------|
| **mi-** | 前 | 主事焦点(AF)，动态性最高 | Ch3,6 |
| **ma-** | 前 | AF(静态)或PF(已发生) | Ch3,6 |
| **<om>** | 中 | AF，过程性动作 | Ch3,6 |
| **∅** | — | AF，少数静态动词 | Ch6 |
| **-en** | 后 | PF，有定受事/终点性 | Ch3,6 |
| **sa-** | 前 | 工具施用(属PF) | Ch6 |
| **-an** | 后 | 处所施用(属PF) | Ch6 |
| **pa-** | 前 | 使动（直接/间接） | Ch3,9 |
| **pa-pi-** | 前+前 | 弱使动（"请/叫...做"） | Ch9 |

### 名物化词缀

| 词缀 | 功能 | 例 |
|------|------|-----|
| **pi-** | 行为名物化 | pi-cocok "打针(的行为)" |
| **ka-** | 语义相关名物化 | ka-etip "西边" |
| **-ay** | 施事名物化("…的人/物") | mifoting-ay "渔夫" |
| **pi-...-an** | 处所名物化 | pi-codad-an "学校" |

### 其他派生前缀

| 词缀 | 语义 | 例 |
|------|------|-----|
| **ci-** | 拥有/生长/穿戴 | ci-payso "有钱", ci-riko' "穿衣服" |
| **hali-** | 喜爱 | hali-^epah "爱喝酒" |
| **sa-** | 製作/像…/当作/加强/工具 | sa-^epah "酿酒", sa-asik "扫把" |
| **tala-** | 前往 | tala-lotok "去山上" |
| **paka-** | 能够 | paka-'araw "能够看到" |

## 3. 焦点标记变换表（跨句型）

| AF叙述 | AF祈使 | AF否定 | PF叙述 | PF祈使 | PF否定 |
|---------|---------|---------|---------|---------|--------|
| mi- | **pi-** | **pi-** | ma- | **ka-** | **ka-** |
| ma- | **ka-** | **ka-** | -en | **-en** | **(ka-)**...**-en** |
| <om> | **ka-<om>** | **ka-<om>** | sa- | **sa-...-en** | **ka-sa-** |
| tayra(∅) | **ka-**tayra | **ka-**tayra | -an | — | **ka-...-an** |

变换核心规则：[FACT] 否定/祈使会将AF标记替换为**非限定形式**(pi-/ka-)。

## 4. TAM标记总汇

| 标记 | 类型 | 功能 | 位置 |
|------|------|------|------|
| **na-** | 前 | 过去/相对过去 | 焦点前 |
| **-ay** | 后 | 实现（进行/已过去） | 词根后 |
| **Ca重疊** | 重疊 | 非实现（未来/从未） | 词首 |
| **=to** | 依附词 | 完成貌 "了" | 动词后 |
| **=ho** | 依附词 | 未完成貌 "还(未)" | 动词后 |

### Ca重叠的多重功能

| 语境 | 功能 | 章 |
|------|------|-----|
| 数词前 (ta-tosa) | "…个人" | Ch3 |
| 动词前 (ma-mi-dotoc) | 非实现/未来 | Ch6 |
| ma-Ca-词根 (ma-ca-coli) | 交互动作 | Ch16 |
| 关系子句中 | 非实现貌 | Ch14 |

## 5. pa-使动系统

| 词根类别 | pa-语义 | 章节 |
|---------|---------|------|
| 名词 | "使有/使增加"：dateng→pa-dateng"加菜" | Ch9 |
| 形容词 | "使变成"：kaeped→pa-kaeped"使变软" | Ch9 |
| 动词 | "使做动作"：kaen→pa-kaen"喂" | Ch9 |
| 交易动词 | 改变交易方向：cakay"买"→pacakay"卖" | Ch9 |

[FACT] 静态动词使动也可用 mi- 或 -en（不加 pa-）：mi-lasang"使醉"，fa'edet-en"加热"。
