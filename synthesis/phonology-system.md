# 阿美语语音系统总览

> 来源：第2章 | 综合日期：2026-05-19

### 元音（5字母/4音位）

| 字母 | 音位(IPA) | 发音 | 分布限制 |
|------|----------|------|---------|
| i | /i/ | ≈中文ㄧ | 可出现在词首/中/尾 |
| u / o | /ʊ/ | 自由变体。近[x/'/h]且同音节时→[o] | 可出现在词首/中/尾 |
| a | /a/ | ≈中文ㄚ | 可出现在词首/中/尾 |
| e | /ə/ | ≈中文ㄜ | **不出现在词尾** |

[FACT] /u/和/o/为自由变体，无辨义作用。北部方言习惯用u，其他方言o/u混用。日语借词多用u。

## 2. 音节结构

```
最大音节：CVVC  (4个语音)
├── C: 声母辅音（至多一个）
├── VV: 韵核 = 复元音（实际为 VG 或 GV 组合）
└── C: 韵尾辅音（至多一个）
```

| 音节类型 | 出现位置 | 例词 |
|---------|---------|------|
| V | — | i.na "妈妈", ta.o "别人" |
| VC | — | im.pic "铅笔" |
| CV | — | fa.fa.hi "太太", ma.ta "眼睛" |
| CVC | — | tam.daw "人", ngo.yos "嘴巴" |
| CVV(CGV) | — | a.cya "赞叹声" |
| CVV(CVG) | — | paw.li "香蕉" |
| CVVC(CGVC) | — | mi.cyan "煎" |

[FACT] 音节一般以辅音开始。元音不能单独构成音节（V型仅出现在特定位置）。

## 3. 重音系统

| 语境 | 规则 |
|------|------|
| 基本规则 | 重音落在词的**最后一个音节** |
| 加词缀后 | 重音**随之后移**，仍落在新词最后音节（如加 -an/-en 时） |
| 是非问句 | 倒数第二音节↗升高，最后音节降至平调；重音移至倒数第二音节 |
| 强调语气 | 倒数第二音节元音延长，重音移至该音节 |
| 叙述句语调 | 下降调 (↘) |

## 4. 七条音韵规则

| # | 规则 | 公式 | 触发条件 | 例 |
|---|------|------|---------|-----|
| 1 | 腭化 | /ts, s/ → [tɕ, ɕ] / __ /i/ | c, s在i前 | posi [pʊɕi]; cima [tɕima] |
| 2 | 辅音送气 | /ʡ/ → [ʡħ] / __ # | '在词尾 | mafoti' [mafʊtiʡħ] |
| 3 | 元音→滑音 | /i, o/ → [j, w] / V__ | 高元音前有另一元音 | maisi' [majsiʡħ] |
| 4 | 舌位降低 | /i, a, o/ → [e, a, ɔ] / /ʡ, h/ __ | 元音在'或h后 | fo'is [fʊʡes]; 'amis [ʡamis] |
| 5 | 滑音加插 | /i, o/ → [ij, ow] / __ V | i/o后接另一元音 | fafahian [fafahejan] |
| 6 | 元音删除 | /ə/ → ∅ / C__ (非重音节) | e在非重音节 | tefo' [tfoʡħ] |
| 7 | 元音延长 | V́# → V́:V# (/强调) | 强调时倒数第二音节 | inacila → [inatɕi:ɾa] |

## 5. 书写系统关键原则

- 书写符号反映基本音位，略去发音细节
- ' (咽头塞音) 和 ^ (喉塞音) 需区分。李壬癸(1992)建议改用 q 表咽头塞音
- o/u 建议各方言择一使用
- 长元音(:)非音位 → 不纳入书写系统
- 专有名词首字母大写，但 ' 和 ^ 之后不加大写（如 'amis 不是 'Amis）
