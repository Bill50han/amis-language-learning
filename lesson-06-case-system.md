# 第6课：格位标记系统全景——阿美语的"语法骨架"

## 本课目标

学完本课后，你将能够：
1. 背出全部 9 个格位标记（主格/属格/斜格 × 普通/专有单/专有复）
2. 理解每个格位的核心功能和使用场景
3. 在简单句子中正确选择格位标记
4. 理解格位标记与焦点系统的呼应关系（初步）
5. 区分格位标记和名词类别标记（O / ci / ca）

## 核心知识

### 1. 为什么需要格位系统？

[FACT] 阿美语的格位标记相当于一套"语法标签"，告诉听者句子中每个名词的语法角色——谁在做动作、谁是动作目标、谁拥有某物。pp.41-42

[COMPARISON] 汉语没有格位标记，完全靠语序（"狗咬人"≠"人咬狗"）。日语有格助词（が/を/に/の），功能类似阿美语的格位标记。阿美语的格位系统比日语更简洁——只有三种格位，但有**名词类型区分**（普通 vs 人称专有）。

### 2. 格位标记总表

[FACT] 阿美语是**三分格位格局**。标记系统按名词类型分为两套，人称专有再分单/复数：p.42

```
┌──────────────────────────────────────────────────────┐
│              格位标记系统 (Case Marking System)        │
├────────────┬──────────┬──────────┬──────────────────┤
│  名词类型   │ 主格 NOM │ 属格 GEN │   斜格 OBL       │
├────────────┼──────────┼──────────┼──────────────────┤
│ 普通名词    │   ko    │   no    │      to          │
├────────────┼──────────┼──────────┼──────────────────┤
│ 专有-单数   │   ci    │   ni    │  ci ...-an       │
├────────────┼──────────┼──────────┼──────────────────┤
│ 专有-复数   │   ca    │   na    │  ca ...-an       │
└────────────┴──────────┴──────────┴──────────────────┘
```

**记忆口诀**：
- 普通名词：k-**o** / n-**o** / t-**o**（韵母都是 o）
- 专有单数：c-**i** / n-**i** / c-**i**...-an（韵母都是 i）
- 专有复数：c-**a** / n-**a** / c-**a**...-an（韵母都是 a）

[FACT] 有趣的是，格位标记深层分析包含名词类别信息：k-o = 主格(k) + 普通名词标记(o)。n-o = 属格(n) + 普通名词标记(o)。这个规律使得标记系统非常对称。（Huang 1995）p.42

### 3. 主格标记（ko / ci / ca）——"句子的主语"

[FACT] 主格标记标示句子的**主语名词组**——与动词焦点词缀构成呼应关系的论元。这是阿美语中**最重要的格位**。pp.42-43

| 功能 | 例句 | 说明 | 来源 |
|------|------|------|------|
| **AF句主事者** | Komaen **ko** singsi to fonga. | 老师在吃地瓜。 | pp.33, 42 |
| **PF句受事者** | Makaen no singsi **ko** fonga. | 地瓜被老师吃了。 | pp.33, 42 |
| **名词谓语句主语** | O singsi **ci** Mayaw. | Mayaw是老师。 | pp.34, 42 |
| **存在句主语** | Ira **ko** fo'is i kakayayan. | 天上有星星。 | pp.75, 42 |
| **工具施用主语** | Sapicekiw ni ina **ko** rinom. | 妈妈用针挑挖。 | pp.65, 42 |
| **处所施用主语** | Pitangtangan ni ina **ko** sefi. | 妈妈在厨房煮。 | pp.67, 42 |

[FACT] **ko 可以标示定指或非定指**。如 Ira **ko** fo'is i kakayayan. "天上有星星"——这里的 ko fo'is 不定指（不是特定哪颗星星）。p.43

[FACT] 主格标记与动词的焦点词缀构成**呼应**：AF 动词 → 主格标记主事者；PF 动词 → 主格标记受事者。pp.42-43

### 4. 属格标记（no / ni / na）——"的"和"被谁"

[FACT] 属格标记两个主要功能：pp.43-44

#### 功能1：领属——"谁的"

| 阿美语 | 结构 | 汉语 | 来源 |
|--------|------|------|------|
| payso **no** singsi | 钱 + no + 老师 | 老师的钱 | p.43 |
| wawa **ni** Panay | 孩子 + ni + Panay | Panay的孩子 | p.43 |
| loma' **na** Panay | 房子 + na + Panay(们) | Panay家的房子 | p.43 |

**公式**：`所有物 + no/ni/na + 所有者`

#### 功能2：非AF句的主事者——"被谁做"

| 阿美语 | 结构 | 汉语 | 来源 |
|--------|------|------|------|
| Makaen **no** singsi ko fonga. | PF.吃 + no+老师 + ko+地瓜 | 地瓜**被老师**吃了。 | p.43-44 |
| Malangto **no** namal koya loma'. | PF.烧毁 + no+火 + 那+房子 | 那间房子**被大火**烧成灰烬了。 | p.44 |

[FACT] **关键规则**：在主事焦点（AF）句中，主事者用主格（ko/ci）；在非主事焦点（PF等）句中，主事者改用属格（no/ni）。p.44

这就是格位和焦点的互动——同一个语义角色（"吃的人"），在不同焦点结构下用不同的格位标记。这将在第9课深入学习。

### 5. 斜格标记（to / ci...-an / ca...-an）——"其余一切"

[FACT] 斜格是功能最杂的格位，涵盖多种非主语、非所有者的语法角色。pp.44-47

| 功能 | 标记 | 例句 | 来源 |
|------|------|------|------|
| **AF句受事者** | to | Komaen ci Mayaw **to** fonga. | p.44 |
| **接受者/受动者** | to / ci...-an | Pafeli ci Kolas to fonga **ci inaan** nira. "Kolas送地瓜给他妈妈。" | p.45 |
| **时间** | to | **to** picocok no ising "当医生打针时" | pp.45-46 |
| **原因** | to | — | p.45 |
| **地点** | **-an**（+ i）| Mikadafo ci Mayaw i **ci Panayan**. "Mayaw入赘到Panay家。" | p.46 |
| **工具**（非主语时）| to | — | p.44 |

[FACT] **重要限制：to 不能标示地点。** 地点用介系词 **i** "在…"。这是初学者的常见错误。p.46

```
✓ Mikadafo ci Mayaw i ci Panayan.     "Mayaw入赘到Panay家。"
✗ Mikadafo ci Mayaw to ci Panayan.    （错误：to不能表地点）
```

[FACT] 早期文献曾将斜格分析为"方位格"或"受格"，但这两种标签都无法全面涵盖其功能，因此本书采用"斜格"这个更中性的术语。p.46

### 6. 名词类别标记 O / ci / ca —— 不是格位！

[FACT] 除了三种格位标记，还有**名词类别标记** O、ci、ca——它们标示名词谓语而非格位功能。p.47

| 标记 | 功能 | 例句 |
|------|------|------|
| **O** | 标示普通名词为谓语 | **O** singsi ci Mayaw. "Mayaw是老师。" |
| **ci** | 标示专有名词单数为谓语 | — |
| **ca** | 标示专有名词复数为谓语 | **Ca** Panay kora pikadafoan. "Panay那家是入赘的。" |

> **区分要诀**：名词类别标记（O）后面是**谓语**（描述），格位标记（ko）后面是**主语**（被描述）。

## 句型公式卡片

```
┌─────────────────────────────────────────────────────┐
│  格位标记速查                                        │
│                                                     │
│  表达"X的Y"        → Y no/ni/na X                   │
│  表达"X做Y"(AF)    → V ko/ci X to Y                  │
│  表达"Y被X做"(PF)  → V no/ni X ko/ci Y               │
│  表达"在L"         → i + 处所名词                     │
│  表达"X是Y"        → O Y ko/ci X                     │
│                                                     │
│  ⚠ to 不能表地点，地点用 i                            │
│  ⚠ PF句中主事者用 no/ni，不用 ko/ci                   │
└─────────────────────────────────────────────────────┘
```

## 例句

| 阿美语 | 汉语 | 格位标注 | 来源 |
|--------|------|---------|------|
| Komaen **ko** singsi **to** fonga. | 老师在吃地瓜。 | ko=主格, to=斜格 | p.33 |
| Makaen **no** singsi **ko** fonga. | 地瓜被老师吃了。 | no=属格, ko=主格 | p.33 |
| Payso **no** singsi kora. | 那是老师的钱。 | no=属格(领属) | p.43 |
| O singsi **ci** Mayaw. | Mayaw是老师。 | O=名词类别, ci=主格 | p.34 |
| Pafeli ci Kolas **to** fonga **ci inaan** nira. | Kolas送地瓜给他妈妈。 | to=斜格(受事), ci...an=斜格(接受者) | p.45 |
| Mikadafo ci Mayaw i **ci Panayan**. | Mayaw入赘到Panay家。 | i+ci...an=处所斜格 | p.46 |
| Wawa **ni** Panay kora. | 那是Panay的孩子。 | ni=属格(领属) | p.43 |

## 语言对比 [COMPARISON]

| 特征 | 阿美语 | 汉语 | 日语 |
|------|--------|------|------|
| 格位系统 | 三分（主/属/斜） | 无 | 多格（が/を/に/の/で…） |
| 名词类型区分 | 有（普通/专有单/专有复） | 无 | 无 |
| 主语显性标记 | 有（ko/ci/ca） | 无 | 有（が/は） |
| 格位-焦点互动 | 有（核心特征） | 无 | 无 |
| 领属标记 | no/ni/na | 的 | の |

### 3. 选择合适的格位标记

填入 ko / no / to / ci / ni / O / i：

| 句子 | 填空 |
|------|------|
| Komaen \_\_ Mayaw \_\_ fonga. | ci, to |
| \_\_ singsi \_\_ Mayaw. | O, ci |
| Makaen \_\_ Mayaw \_\_ fonga. | ni, ko |
| Wawa \_\_ Panay kora. | ni |
| Mikadafo ci Mayaw \_\_ \_\_ Panayan. | i, ci |

### 4. 改错

| 错误 | 改正 | 原因 |
|------|------|------|
| Komaen to Mayaw ci fonga. | Komaen ci Mayaw to fonga. | 人名Mayaw是主事者，应用ci；地瓜fonga是受事者，应用to |
| Mikadafo ci Mayaw to ci Panayan. | ? | to不能表地点，改用i |
| Makaen ko Mayaw ko fonga. | ? | PF句主事者应用ni，受事者用ko |
| Singsi ko Mayaw. | ? | 名词谓语句缺O |
