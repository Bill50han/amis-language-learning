# 第9课：焦点系统入门——AF vs PF

## 本课目标

学完本课后，你将能够：
1. 理解"焦点"（focus）在阿美语中的核心概念
2. 区分主事焦点（AF）和受事焦点（PF）的句子
3. 看懂格位标记如何随焦点变化而改变
4. 理解为何 AF 句"像主动句"、PF 句"像被动句"——但不完全等同
5. 理解为什么焦点系统是阿美语的"语法心脏"

---

## 核心知识

### 1. 问题引入：两种说法，一个事件

[FACT] 在阿美语中，同一个事件可以用**两种不同的句法结构**来表达：pp.33, 59-60

| 版本 | 阿美语 | 汉语翻译 |
|------|--------|---------|
| **AF句** | K**\<om\>**aen **ci** Mayaw **to** fonga. | Mayaw吃地瓜。 |
| **PF句** | Ma**kaen** **ni** Mayaw **ko** fonga. | 地瓜被Mayaw吃了。 |

两个句子的意思基本相同——"Mayaw吃地瓜"这个事件。但语法结构完全不同：

```
AF句:  Komaen       ci Mayaw    to fonga.
      吃·AF(主事焦点) 主格-主事者   斜格-受事者
      ↑主语=吃的人           ↑宾语=被吃的东西

PF句:  Makaen       ni Mayaw    ko fonga.
      吃·PF(受事焦点) 属格-主事者   主格-受事者
                      ↑"被谁"     ↑主语=被吃的东西
```

**核心差异**：哪个参与者当**主语**（被格位标记 ko/ci 标示的那个）。

- AF 句：**主事者**（吃的人）当主语
- PF 句：**受事者**（被吃的东西）当主语

### 2. 什么是"焦点"？

[FACT] "焦点"（focus）是阿美语动词上的标记系统，其功能是**告诉听者：句子主语扮演什么语义角色**。pp.59-60

```
┌──────────────────────────────────────────────────┐
│  焦点系统 = 在动词上标记"哪种参与者在当主语"       │
│                                                  │
│  AF标记 → 主语 = 主事者（做动作的人/经历者）        │
│  PF标记 → 主语 = 受事者（承受动作的人/物）          │
│               或 = 工具（用sa-）                  │
│               或 = 处所（用-an）                  │
└──────────────────────────────────────────────────┘
```

[COMPARISON] 这个概念在汉语和英语中没有直接对应。最接近的理解方式：

- **AF ≈ "主动"视角**——突出做动作的人
- **PF ≈ "被动"视角**——突出承受动作的东西

但这不是主动/被动的简单对立。因为阿美语中，AF/PF 的选择不像英语主动/被动那样有"降级"或"语用特殊"的意味——两种都是常用的、自然的表达方式。

### 3. 同一个动词的 AF 和 PF 形式

[FACT] 同一个动词词根可以通过不同焦点标记，变成 AF 或 PF 形式：pp.59-64

| 词根 | AF 形式 | PF 形式 | 汉语 |
|------|---------|---------|------|
| **kaen** "吃" | k**\<om\>**aen | ma**kaen** | 吃 |
| **foting** "鱼" | **mi**foting | — | 捕鱼 |
| **'atal** "盛" | **mi**'atal | — | 盛 |
| **cikay** "跑" | c**\<om\>**ikay | — | 跑（不及物，无PF） |
| — | — | **-en** (后缀) | PF另一标记 |

[FACT] AF 句在语法上是**不及物结构**（主语=主事者），PF 句才是**及物结构**（主语=受事者，另有属格标记者）。p.61

这不是直觉能想到的——AF 句看起来有"宾语"（带 to），但语法上算不及物！这体现了阿美语的语法逻辑：**只有 PF 句才是真正的及物结构**。

### 4. AF 句和 PF 句的结构对比

[FACT] 最直观的对比是通过同一个事件的两种表达：pp.33, 59-60

```
┌─────────────────────────────────────────────────────┐
│  AF句（主事焦点）                                    │
│  ┌─────────┬─────────────┬──────────┐               │
│  │ V.AF    │ 主事者(主格) │ 受事者(斜格) │               │
│  │ 焦点在   │ ko/ci 标记   │ to 标记    │               │
│  │ 主事者   │ = 主语       │ = 非主语   │               │
│  └─────────┴─────────────┴──────────┘               │
│  例: Komaen   ci Mayaw      to fonga.               │
│       "Mayaw吃地瓜"                                 │
├─────────────────────────────────────────────────────┤
│  PF句（受事焦点）                                    │
│  ┌─────────┬─────────────┬──────────────┐           │
│  │ V.PF    │ 主事者(属格) │ 受事者(主格)   │           │
│  │ 焦点在   │ no/ni 标记   │ ko/ci 标记    │           │
│  │ 受事者   │ = "被谁"    │ = 主语        │           │
│  └─────────┴─────────────┴──────────────┘           │
│  例: Makaen   ni Mayaw      ko fonga.               │
│       "地瓜被Mayaw吃了"                              │
└─────────────────────────────────────────────────────┘
```

**关键规律**：

| 要素 | AF 句 | PF 句 |
|------|-------|-------|
| 动词标记 | AF 标记（mi-/ma-/\<om\>） | PF 标记（ma-/-en） |
| 主事者格位 | **主格**（ko/ci） | **属格**（no/ni） |
| 受事者格位 | **斜格**（to） | **主格**（ko/ci） |
| 哪个是主语？ | 主事者 | 受事者 |

这就是第6课提到的"格位-焦点呼应"——动词上的焦点标记决定了哪个名词带主格。

### 5. 什么时候用 AF？什么时候用 PF？

[FACT] 教材未给出严格的使用条件规则。pp.59-60

[INFERENCE] 根据教材例句分析，AF 与 PF 的使用倾向如下：

| 场景 | 倾向 | 原因 |
|------|------|------|
| 主事者是已知信息，受事者是新信息 | AF | "谁做的"更重要 |
| 受事者是话题/已知信息，关注结果 | PF | "东西怎么样了"更重要 |
| 主事者是无生命物 | PF（ma-） | ma- PF 允许无生命主事者 |
| 强调动作的意志性、过程 | AF | mi- 最具动态性 |
| 关系子句中 | 取决于先行词角色 | 第14课详述 |

[FACT] 选择 AF 还是 PF 由**语用和信息结构**决定，类似汉语选择"把字句"vs"被字句"vs 普通句——不是随意的，但也不是完全机械的。pp.59-60

### 6. 本书的创新：两焦分析

[FACT] 传统分析认为阿美语有四种焦点：主事焦点(mi-/ma-/<om>)、受事焦点(ma-/-en)、工具焦点(sa-)、处所焦点(-an)。

本书采用**两焦分析**：只分**主事焦点(AF)**和**受事焦点(PF)**。工具标记(sa-)和处所标记(-an)被重新分析为 PF 之下的**施用结构**（applicative）。pp.59-60

关键证据：
1. 工具/处所标记可与 PF 标记共现（如 ma-sa-pi-sanga'）
2. 在关系子句中，工具/处所动词不需要时貌变化，但 AF/PF 必须
3. 工具/处所没有明显的时貌讯息

对学习者来说，这意味着：**先理解 AF vs PF 的基本对立**，工具/处所是 PF 的延伸用法。


## 语言对比 [COMPARISON]

| 特征 | 阿美语 | 汉语 | 英语 |
|------|--------|------|------|
| 语态系统 | **焦点系统**（AF/PF对称） | 主动/"被"字句（不对称） | 主动/被动（不对称） |
| AF ≈ | 动词标记不同的主动句 | 普通主动句 | active voice |
| PF ≈ | 受事当主语的及物句 | "被"字句 / 受事主语句 | passive voice |
| 选择自由度 | 高——两种都是自然表达 | 低——"被"字句有特殊语用色彩 | 低——passive有特殊语用色彩 |
| 动词变化 | 焦点词缀变化 | "被"+动词（无词形变化） | be + past participle |

[FACT] 阿美语的 AF/PF 比汉语主动/被动更**对称**——两种结构都是核心语法，没有"被动是特殊形式"的意味。pp.59-60

[INFERENCE] 基于教材中 PF 句的自然使用频率和中性语义特征，PF 句的使用频率远高于汉语的"被"字句。教材未做定量对比，此结论根据教材例句分布推断。

### 4. 简答

- **为什么 AF 句在语法上算"不及物"？**  
  提示：AF 句的主事者是主格（主语），受事者是斜格（非核心论元）——句子语法上只有一个核心论元（主语），所以是不及物结构。

- **同一个事件，AF 和 PF 的语义区别是什么？**  
  提示：不是事件本身变了，而是"视角"变了——说话人选择让哪个参与者当主语。
