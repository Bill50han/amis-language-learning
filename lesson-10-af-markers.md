# 第10课：AF标记详述——mi- / ma- / \<om\> / ∅

## 本课目标

学完本课后，你将能够：
1. 掌握四种 AF 标记的动态性阶梯和语义差异
2. 根据语义需求选择正确的 AF 标记
3. 理解 AF 标记与及物性/不及物性的关系
4. 区分 mi- 和 <om>——两者都表动作，有何不同？
5. 理解为何 ma- 动词既可以是 AF 也可以是 PF

---

## 核心知识

### 1. 回顾：AF 标记总览

[FACT] 主事焦点（AF）有四种标记形式，按**动态性**（动作的力度、施事的控制力）从高到低排列：pp.61-63

```
动态性高 ←──────────────────────────→ 动态性低

  mi-          <om>           ma-           ∅
  │             │              │             │
  主动动作      过程/姿态      状态/经历      属性
  "捕鱼"       "在哭"        "饿/聪明"     "高"
  及物/不及物   大多不及物     大多不及物     不及物
```

[FACT] 四种标记的共同点：主语 = 主事者（动作执行者 / 经历者 / 状态持有者）。pp.61-63

### 2. mi- —— 最具动态性

[FACT] mi- 是动态性最高的 AF 标记。标示主动、有意的、由施事控制的行为。pp.61-62

| 特征 | 内容 |
|------|------|
| 动态性 | 最高——施事有意志力、有控制 |
| 典型语义 | 主动动作（"捕鱼""盛饭""学习""洗澡"） |
| 及物性 | 可以是及物（带 to 受事者）或不及物 |
| 动作性 | 动作本身在施事控制之下 |

**常见 mi- 动词**：

| 动词 | 词根 | 词根原义 | 汉语 |
|------|------|---------|------|
| **mi**-foting | foting | 鱼 | 捕鱼 |
| **mi**-'atal | 'atal | 盛装 | 盛（饭） |
| **mi**-nanam | nanam | 学习 | 学习 |
| **mi**-liloc | liloc | 洗澡 | 洗澡 |
| **mi**-kayakay | kayakay | 爬 | 爬 |
| **mi**-cekaw | cekaw | （乐器名） | 弹奏乐器 |

**及物用法（带 to）**：

| 阿美语 | 汉语 |
|--------|------|
| Mi'atal kako **to** hemay. | 我在盛饭。 |
| Minanam kiso **to** Amirang a sowal. | 你在学阿美语。 |

**不及物用法（不带 to）**：

| 阿美语 | 汉语 |
|--------|------|
| Mililoc kako. | 我在洗澡。 |
| Mifoting ci Mayaw. | Mayaw在捕鱼。 |

[FACT] mi- 动词大多从名词派生：foting "鱼" → mi-foting "捕鱼"；'atal "盛装" → mi-'atal "盛饭"。这是一种常见的构词模式。pp.61-62

[COMPARISON] mi- 动词的构词方式类似汉语的"名动转类"（"鱼"→"渔"），但阿美语通过系统的 mi- 前缀来实现，比汉语更规律。

### 3. \<om\> —— 过程性、姿态性

[FACT] <om> 是一个**中缀**（插入词根首辅音之后），标示中等动态性的动作。动作有一定过程性，但施事的意志力和控制力不如 mi- 动词强。pp.62

| 特征 | 内容 |
|------|------|
| 动态性 | 中等——有过程性、常为身体动作/姿态 |
| 典型语义 | 姿态、生理过程、表达性动作 |
| 及物/不及物 | 少数可及物（如 komaen "吃"） |

**<om> 插入规则**：

```
词根首辅音 + om + 词根其余部分
```

| 词根 | 插入 <om> | 汉语 |
|------|----------|------|
| **t**angic | t**\<om\>**angic | 在哭 |
| **r**adiw | r**\<om\>**adiw | 唱歌 |
| **c**ikay | c**\<om\>**ikay | 在跑 |
| **k**aen | k**\<om\>**aen | 吃 |
| **s**a'ic | s**\<om\>**a'ic | 编制（鱼网） |

[FACT] <om> 不出现在元音开头的词根中。p.62

#### mi- vs <om> —— 两者的微妙区别

[FACT] 教材指出 mi- 型"最具动态性"，<om> 型"次之"，但没有给出更精确的区分规则。p.62

[INFERENCE] 从教材例句推断：
- **mi-** 强调动作的**主动性、目的性、结果导向**（如"捕鱼"是有目的的）
- **<om>** 强调动作的**过程性、自发性**（如"哭"是自己发生的，不是刻意去做的）

部分动词似乎两种标记都可（如 komaen vs mikaen"吃"），原因可能是 [UNCERTAIN] 方言或语体差异。

[COMPARISON] 类似英语中 run（简单动词） vs be running（进行体）的区别线索，但不完全一致。阿美语的区别更多在于动作的**控制度**和**动态性本质**。

### 4. ma- (AF) —— 状态、被动经历

[FACT] ma- 在 AF 句中标示**低动态性**——静态属性、不由自主的状态、或非有意控制的经历。pp.62-63

| 特征 | 内容 |
|------|------|
| 动态性 | 低——施事不控制、不主动 |
| 典型语义 | 状态（"饿""聪明""怕"）、被动经历（"跌倒""受伤"） |
| 及物性 | 几乎都是不及物 |
| 主语角色 | **经历者**而非执行者 |

**常见 ma- AF 动词**：

| 动词 | 汉语 | 为什么是 ma-？ |
|------|------|--------------|
| **ma**-cahiw | 肚子饿 | 饿不是你"主动做的" |
| **ma**-foti' | 睡觉 | 睡觉是自然状态 |
| **ma**-taneng | 聪明 | 聪明不是你"做的" |
| **ma**-talaw | 害怕 | 怕是不由自主的 |
| **ma**-olah | 喜欢/爱 | 爱不是你能控制的 |
| **ma**-tolo' | 跌倒 | 跌倒不是故意的 |
| **ma**-hadidi | 忍耐 | — |

**例句**：

| 阿美语 | 汉语 | 来源 |
|--------|------|------|
| Macahiw kako. | 我肚子饿。 | p.35 |
| Mataneng ko wawa. | 孩子很聪明。 | p.62 |
| Matalaw kako to waco. | 我怕狗。 | — |
| Maolah kako i tisowanan. | 我爱你。 | — |
| Matolo' ci Mayaw. | Mayaw跌倒了。 | p.62 |

[FACT] ma- 动词的主语是**经历者**（experiencer），不是主动的动作执行者。这是 ma- AF 和 mi- AF 最核心的区别。p.62

#### ⚠ 注意：ma- 也可以是 PF！

[FACT] 同一个 ma- 形式，在不同句法环境中可以是 AF 也可以是 PF：p.63

| ma- 的用法 | 例句 | 判断 |
|-----------|------|------|
| AF 静态 | **Ma**taneng ko wawa. | AF——不及物，主语=经历者 |
| AF 被动经历 | **Ma**tolo' ci Mayaw. | AF——不及物，主语=经历者 |
| PF 受事 | **Ma**kaen no singsi ko fonga. | PF——及物，主语=受事者 |

**区分的关键**：看句子中是否有带**属格**（no/ni）的主事者。
- 有 no/ni → ma- 是 PF（及物，受事焦点）
- 没有 → ma- 是 AF（不及物，主事焦点）

### 5. ∅ 零标记 —— 固有属性

[FACT] 少数静态动词/形容词不带任何焦点标记，直接用词根做谓语。pp.63

| 动词 | 汉语 |
|------|------|
| Takaraw | 高 |
| Adada | 痛 |
| Tatiih | 坏 |

| 阿美语 | 汉语 |
|--------|------|
| Takaraw to kira wawa. | 那孩子长高了。 |
| Adada ko mata nira. | 他的眼睛痛。 |

这类词数量很少，遇到时个别记忆即可，不需要特别的构词规则。

## AF 标记选择流程图

```
你需要表达一个状态/事件，主语是动作的发出者/经历者

├─ 主语主动、有意志地做某事？ → mi-
│   例：捕鱼、盛饭、学习、洗澡
│
├─ 主语经历过程/姿态、不太有控制？ → <om>（中缀）
│   例：哭、唱歌、跑
│
├─ 主语经历不由自主的状态/被动经历？ → ma-
│   例：饿、聪明、害怕、爱、跌倒
│
└─ 主语有固有属性？（极少数词） → ∅
    例：高、痛
```

## 语言对比 [COMPARISON]

| AF标记 | 语义 | 汉语类比 | 英语类比 |
|--------|------|---------|---------|
| mi- | 主动动作 | "去+动词"（去捕鱼） | go fishing / do V |
| <om> | 过程/姿态 | 零标记动词（"哭""跑"） | 简单动词（cry, run） |
| ma- | 状态/经历 | 形容词（"饿""聪明"）/ "被"隐含 | be + adj / feel |
| ∅ | 固有属性 | 形容词（"高""痛"） | be + adj |

[FACT] 这四种标记不是任意的。它们系统性地反映动词的**动态性**和**主语的控制度**。pp.61-63
