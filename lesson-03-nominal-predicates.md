# 第3课：名词谓语句——"X是Y"怎么说

## 本课目标

2. 用 `O ... ko ...` 结构说"X是Y"型的句子
3. 区分 ko（普通名词主格）和 ci（专有人名主格）
4. 区分 O 和 ko 的不同功能
5. 说出至少 5 个常用职业/身份名词

## 核心知识

### 1. 阿美语的第一原则：谓语在前

[FACT] 阿美语是**谓语在前（predicate-initial）**的语言。句子分为"谓语"（描述部分）和"主语"（被描述的对象），谓语永远在主语前面。p.33

```
句子 = 谓语 + 主语（+ 其他成分）
      ↑描述     ↑被描述
```

这跟汉语刚好相反。比较：

| 语言 | 结构 | "Mayaw是老师" |
|------|------|-------------|
| 汉语 | 主语 → 谓语 | Mayaw **是** 老师。 |
| 阿美语 | 谓语 → 主语 | **O** singsi **ci** Mayaw. |

[COMPARISON] 汉语是 SVO 语言，"主语-动词-宾语"；阿美语是谓语在前语言，"谓语-主语"。这个差异贯穿整个阿美语语法，是最根本的结构特征。初次接触可能会觉得"倒过来了"，但这是学习的第一个也是最需要适应的习惯。

### 2. 名词谓语句：基础公式

[FACT] 当谓语是名词时（"X是Y"），结构如下：p.34

```
O  +  名词谓语  +  ko/ci  +  主语
│       │           │         │
名词    描述       主格      被描述
类别    身份/      标记      的对象
标记    属性
```

#### 2.1 普通名词主语 → 用 ko

| 阿美语 | 结构分析 | 汉语 |
|--------|---------|------|
| **O** singsi **ko** safa. | O + 老师 + ko + 弟妹 | 弟妹是老师。 |
| **O** tamdaw **kako**. | O + 人 + 我(主格) | 我是人。 |
| **O** loma' **koni**. | O + 房子 + 这(主格) | 这是房子。 |

#### 2.2 专有人名/亲属名称主语 → 用 ci

| 阿美语 | 结构分析 | 汉语 |
|--------|---------|------|
| **O** singsi **ci** Mayaw. | O + 老师 + ci + Mayaw | Mayaw是老师。 |
| **O** ising **ci** Panay. | O + 医生 + ci + Panay | Panay是医生。 |
| **O** mama ako **ci** Kacaw. | O + 爸爸 + 我的 + ci + Kacaw | Kacaw是我爸爸。 |

#### 2.3 名词类别标记 ca：专有复数做谓语

| 阿美语 | 结构分析 | 汉语 |
|--------|---------|------|
| **Ca** Panay kora pikadafoan. | ca + Panay(们) + 那个+入赘之处 | 那家入赘的是Panay她们。 |

[FACT] p.47

### 3. O 是什么？ko 是什么？——关键区分

这是本课最重要的概念区分：[FACT] pp.42, 47

| 标记 | 性质 | 功能 | 位置 |
|------|------|------|------|
| **O** | 名词类别标记 | 标记后面的名词是**谓语**（描述部分） | 在谓语名词前 |
| **ko** | 主格格位标记 | 标记后面的名词是**主语**（被描述对象） | 在主语名词前 |
| **ci** | 专有名词主格标记 | 标记后面的人名/亲属是**主语** | 在主语人名前 |

一个形象的记忆方式：

> O 类似汉语"**是**"的位置（但不是动词，是标记）
> ko/ci 类似汉语"**主语**"的标记

```
O    singsi     ci   Mayaw.
│    │          │    │
是-  老师       -主  Mayaw
标记            格   (主语)
(谓语)          标记
```

[FACT] O 的实际功能是告诉听者"接下来的是一个名词谓语"。ko 告诉听者"接下来的是主语"。两者配合完成"X是Y"型的句子。p.34, 47

[COMPARISON] 汉语用"是"（动词）连接主语和谓语名词。阿美语没有系动词"是"，而是用标记 O 和 ko 来分别标记谓语和主语。这类似于日语的「～は～だ」结构，但语法机制不同。

### 4. 否定形式

[FACT] 名词谓语句的否定形式，在 O 前面加 **caay**，并且 O 换成 **ko**：p.34

| 肯定 | 否定 |
|------|------|
| **O** singsi **ci** Mayaw. | **Caay ko** singsi **ci** Mayaw. |
| "Mayaw是老师。" | "Mayaw不是老师。" |

公式：
```
Caay ko + 名词谓语 + ci/ko + 主语
```

更多例句：

| 阿美语 | 汉语 |
|--------|------|
| Caay ko tamdaw kora. | 那不是人。 |
| Caay ko ising ci Mayaw. | Mayaw不是医生。 |
| Caay ko mama ako ci Panay. | Panay不是我妈妈。 |

## 句型公式卡片

```
┌─────────────────────────────────────────────┐
│  肯定： O + 名词谓语 + ko/ci/ca + 主语        │
│  否定： Caay ko + 名词谓语 + ko/ci/ca + 主语   │
│                                             │
│  O    = 名词类别标记（标记谓语名词·普通）         │
│  ci   = 名词类别标记（标记谓语名词·专有单数）/      │
│         专有名词单数主格标记（标记主语）             │
│  ca   = 名词类别标记（标记谓语名词·专有复数）        │
│  ko   = 普通名词主格标记（标记主语）              │
│  caay = 否定词"不/不是"                        │
│                                             │
│  注：ci 有双重功能——谓语标记（=O）和主语标记（=ko）。 │
│  在 O + N + ci + Name 中，ci = 主格；              │
│  在 Ca + Name + ... 中，ca = 名词类别标记。         │
└─────────────────────────────────────────────┘
```

## 语言对比 [COMPARISON]

| 特征 | 阿美语 | 汉语 | 英语 |
|------|--------|------|------|
| 系动词"是" | **无**（用标记 O + ko） | 有（"是"） | 有（be） |
| 语序 | 谓语-主语 | 主语-谓语 | 主语-谓语 |
| 否定"不是" | caay ko | 不是 | is not |
| 名词标记 | O（谓语标记）/ ko（主语标记） | 无 | 无 |
| 人称 vs 普通名词 | 不同标记（ci/ko） | 无区别 | 无区别 |

[FACT] 阿美语没有像汉语"是"或英语 be 这样的系动词。它通过标记系统来区分谓语和主语。p.34
