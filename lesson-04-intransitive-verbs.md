# 第4课：不及物动词谓语句——"我在睡觉""孩子很聪明"

## 本课目标

学完本课后，你将能够：
1. 理解阿美语动词谓语的基本词序：动词在前、主语在后
2. 掌握三类主事焦点（AF）动词标记：mi-、ma-、<om>
3. 区分动态动词和静态动词的标记选择
4. 用不及物动词造单论元句（一个参与者的句子）
5. 积累约 15 个常用不及物动词

---

## 核心知识

### 1. 复习：谓语在前原则

[FACT] 第3课学习的"谓语在前"规则不仅适用于名词谓语，也适用于动词谓语：p.33-35

```
汉语（SVO）:  主语  +  动词  +  (宾语)
            我      吃       (地瓜)

阿美语（V-initial）:  动词  +  主语  +  (其他)
                   Komaen   kako     (to fonga)
                   吃       我        (地瓜)
```

本课先学**单论元句**（只有一个参与者的句子，即不及物动词句）：

```
动词 + 主语
V    + ko/ci/代名词(主格)
```

### 2. 阿美语动词的四类 AF 标记

[FACT] 阿美语的动词携带**焦点标记**（focus marker），表明句子的主语扮演什么角色。本课先学**主事焦点（Actor Focus, AF）**——即主语是动作的**执行者**或状态的**经历者**。pp.61-63

AF 标记有四种，按**动态性**（动作的"力度"）从高到低排列：

| 标记 | 动态性 | 典型语义 | 例词 | 汉译 |
|------|--------|---------|------|------|
| **mi-** | 最高 | 主动动作 | mi-**foting** | 捕鱼 |
| **\<om\>** | 中等 | 过程、姿态 | t**\<om\>**angic | 在哭 |
| **ma-** | 低 | 状态、被动经历 | ma-**cahiw** | 肚子饿 |
| **∅** | 最低 | 属性 | takaraw | 高 |

[FACT] 这四类 AF 标记的共同点是：**主语 = 主事者**（动作者/经验者/属性持有者）。pp.61-63

[COMPARISON] 汉语不区分动词的"动态性"——"我吃饭""我饿""我高兴"的动词没有任何标记区分。阿美语通过前缀（mi-/ma-）或中缀（<om>）明确标示动作性质。这有点像日语「食べる」vs「走る」vs「できる」的动词分类，但是通过语法标记而非词汇本身来区分。

### 3. mi- 动态动词："主动做某事"

[FACT] mi- 是最常见的主事焦点标记，标示**主动、有意的动作**。pp.61-62

**构成**：mi- + 词根

| 阿美语 | 结构 | 汉语 |
|--------|------|------|
| **Mi**foting kako. | mi-捕鱼 + 我 | 我在捕鱼。 |
| **Mi**nanam kako. | mi-学习 + 我 | 我在学习。 |
| **Mi**kaen kako. | mi-吃 + 我 | 我在吃（饭）。 |
| Ro**ma**diw ci Panay. | <om>唱歌 + ci+Panay | Panay在唱歌。 |

等等——romadiw 不是 mi- 动词！它是 <om> 动词（中缀）。让我重新整理。

[FACT] mi- 动词例子：pp.61-62

| 阿美语 | 词根 | 汉语 |
|--------|------|------|
| **mi**-foting | foting "鱼" | 捕鱼 |
| **mi**-'atal | 'atal "盛装" | 盛（饭） |
| **mi**-cekaw | cekaw "一种乐器" | 弹奏乐器 |
| **mi**-nanam | nanam "学习" | 学习 |
| **mi**-liloc | liloc "洗澡" | 洗澡 |

例句：

| 阿美语 | 汉语 | 来源 |
|--------|------|------|
| Mililocay kako. | 我正在洗澡。 | p.72 |
| Mifoting ci Mayaw. | Mayaw在捕鱼。 | — |
| Mi'atal kako to hemay. | 我在盛饭。 | p.61 |

### 4. \<om\> 中缀动词："过程性动作"

[FACT] <om> 是一个**中缀**（infix），插入词根**第一个辅音之后**。标示中等动态性的动作，常涉及过程或姿态。pp.61-62

**构成**：词根首辅音 + **om** + 词根其余部分

| 词根 | 加 <om> 后 | 汉语 | 说明 |
|------|-----------|------|------|
| **t**angic | t**\<om\>**angic | 在哭 | t + om + angic |
| **r**adiw | r**\<om\>**adiw | 唱歌 | r + om + adiw |
| **c**ikay | c**\<om\>**ikay | 跑 | c + om + ikay |
| **k**aen | k**\<om\>**aen | 吃 | k + om + aen |

[FACT] <om> 出现在以辅音开头的词根中。如果词根以元音开头则不适用。p.62

[COMPARISON] 中缀在汉语和英语中几乎不存在（英语偶有如"abso-freaking-lutely"但非语法标记）。阿美语的 <om> 插入在词根内部，这是南岛语的特色之一。藏缅语系也有一些语言有中缀，但汉语完全没有这个概念。

例句：

| 阿美语 | 汉语 | 来源 |
|--------|------|------|
| T\<om\>angic kira wawa. | 那小孩在哭。 | p.62 |
| R\<om\>adiw ci Panay. | Panay在唱歌。 | — |
| C\<om\>ikay kako. | 我在跑步。 | — |
| K\<om\>aen ci Mayaw to fonga. | Mayaw在吃地瓜。 | p.33 |

**注意**：komaen（吃）和 mikaen（吃）都可以——komaen 用 <om>，mikaen 用 mi-。教材中两种形式都出现。原因可能是 [INFERENCE] 方言或语体差异，教材未明确说明选择条件。

### 5. ma- 静态动词："处于某种状态"

[FACT] ma- 标示**低动态性**——静态属性、不由自主的状态、或非有意控制的经历。pp.62-63

**构成**：ma- + 词根

| 阿美语 | 词根 | 汉语 |
|--------|------|------|
| **ma**-cahiw | cahiw "饿" | 肚子饿 |
| **ma**-foti' | foti' "睡" | 睡觉 |
| **ma**-taneng | taneng "聪明" | 聪明 |
| **ma**-talaw | talaw "怕" | 害怕 |
| **ma**-olah | olah "爱" | 喜欢/爱 |
| **ma**-hadidi | hadidi "忍耐" | 忍耐 |
| **ma**-tolo' | tolo' "跌倒" | 跌倒 |

例句：

| 阿美语 | 汉语 | 来源 |
|--------|------|------|
| Macahiw kako. | 我肚子饿。 | p.35 |
| Mafoti' kako. | 我在睡觉。 | — |
| Mataneng ko wawa. | 孩子很聪明。 | p.62 |
| Matalaw kako. | 我害怕。 | — |
| Maolah kako i tisowanan. | 我爱你。 | — |

[FACT] ma- 动词的主语不是主动执行者，而是**经历者**或**状态持有者**。如 Macahiw（饿）是你经历的状态，不是你"主动做的"。pp.62-63

### 6. ∅ 零标记静态动词

[FACT] 少数**静态动词/形容词**不带任何焦点标记，直接用词根作为谓语。p.63

| 阿美语 | 汉语 |
|--------|------|
| Takaraw to kira wawa. | 那孩子长高了。 |
| Adada ko mata nira. | 他的眼睛痛/他有眼疾。 |

这类动词数量较少，在初学阶段遇到时特别记忆即可。

### 7. 小结：如何选择 AF 标记？

```
主动的、故意的动作 ──→ mi-
有过程、姿态性动作 ──→ <om>（中缀）
不由自主的状态/属性 ──→ ma-
少数固有属性形容词 ──→ ∅（零标记）

有疑问时：mi- 的覆盖范围最广（最动态），ma- 对静态/被动经历最明确。
```

## 语言对比 [COMPARISON]

| 特征 | 阿美语 | 汉语 | 英语 |
|------|--------|------|------|
| 不及物动词标记 | 必须带 AF 标记（mi-/ma-/<om>/∅） | 无 | 无 |
| 动态性 | 动词前缀标记动态性高低 | 靠词汇本身/语境 | 靠词汇本身 |
| 施事/经历区分 | ma- 标记经历者主语 | 无标记区分 | 无标记区分 |
| 词序 | V-S | S-V | S-V |

[FACT] 阿美语不及物动词句的主语永远是**主格**（ko/ci/主格代名词）。动词的 AF 标记和 ko 标记形成**呼应**——它们一起告诉听者"主语是动作的执行者/经历者"。pp.61-63
